# Prompt 使用指南

填好 `output/` 中的财务档案后，在 Claude Code 中直接复制以下 prompt 使用。

> 使用前请先确认已按 `templates/README.md` 的说明填写个人档案，然后在 Claude Code 中输入 `/wealth-advise` 或直接粘贴 prompt。

---

## 新手入门（适合小陈这样的情况）

**账户优先级**
```
我刚入职，请根据我的基本信息和财务目标，告诉我应该开哪些账户，优先级怎么排，每年各放多少钱
```

**贷款 vs 投资**
```
我有学生贷款，同时想开始投资建立退休账户。请根据我的收入和贷款利率，分析应该怎么平衡还款和投资
```

**401K 选择**
```
我的 401K 应该选传统（税前）还是 Roth（税后）？请根据我的收入、税率和预期退休年龄分析
```

**报税基础**
```
作为 H1B 工作者，我的报税身份是什么？有哪些常见的可扣除项目我可能漏掉了？
```

## 退休规划（适合阿明这样的情况）

**退休准备度评估**
```
请评估我现在的退休准备度：按我目前的资产和储蓄率，预计何时能达到财务独立？距离目标还差多少？
```

**Roth Conversion 分析**
```
我今年的收入情况请见档案。请分析我是否适合做 Roth Conversion，如果做的话建议转多少金额，在哪个税率区间内最划算
```

**资产配置优化**
```
请根据我的账户余额、年龄和风险偏好，评估当前资产配置是否合理，给出调整建议
```

**退休提取策略**
```
假设我计划 X 年后退休，请制定一个退休账户的提取顺序策略，最小化未来税务负担
```

## 中美双边（适合王女士这样的情况）

**回国财务清单**
```
我持有绿卡，考虑 5 年内回国定居。请列出回国前需要完成的财务事项清单，按优先级排序
```

**FBAR / FATCA 申报**
```
我有中美双边金融账户，请根据我的账户资产情况，说明我的 FBAR 和 FATCA 申报义务，以及关键截止日期
```

**绿卡放弃评估**
```
我已持有绿卡超过 8 年，请分析放弃绿卡的 Exit Tax 风险，我是否符合 Covered Expatriate 条件，有哪些规划可以降低税务影响
```

**中国资产处置**
```
我在中国有房产和存款，请分析回国前如何合规处置或保留这些资产，有哪些税务和外汇合规注意事项
```

## 通用

**整体财务健康诊断**
```
请根据我的四份财务档案，给我一份整体的财务健康诊断，指出最需要关注的 3-5 个方面
```

**年度税务优化**
```
根据我今年的收入和账户情况，列出年底前可以采取的税务优化措施，按节税金额排序
```

**保险覆盖评估**
```
请评估我目前的保险覆盖是否足够，有哪些常见的保险漏洞需要补上
```
