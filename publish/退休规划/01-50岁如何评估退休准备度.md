## 场景

阿明50岁，在硅谷做软件工程师已经20年。账户里有钱，但说不清"够不够"。

他知道自己有三个账户：401K 大约 $800K，Roth IRA 大约 $120K，应税账户大约 $200K。年薪 $250K，两个孩子已经上大学，房贷还有10年，生活开支在 $120K 左右（含税前估算）。

问题来了：

- 我的 $1.12M 总资产算到哪一步了？
- 还需要多少年才能退休？
- 现在应该做什么？

这篇文章演示如何用一套结构化框架来回答这些问题。

---

## 怎么做

评估退休准备度分五步：

### 第一步：算出你的财务独立数字（FI Number）

财务独立数字 = **年支出 × 25**

这个公式来自 [[wiki/退休规划/4%规则\|4% 规则]]：退休后每年提取组合的 4%，历史数据表明这个速率可以支撑至少30年而不耗尽资产。反过来说，你需要积累"年支出的25倍"才能达到财务独立。

> 注意：加州居民提取是税前金额，如果实际生活支出是 $10万，考虑 25–30% 的税负，目标组合规模应按税前支出来算，或者直接把目标提高 30%。

### 第二步：算出当前进度

当前进度 = **总资产 ÷ FI Number**

这个数字告诉你站在哪里，离终点还有多远。

### 第三步：识别退休阶段

[[wiki/退休规划/三段式退休框架\|三段式退休框架]] 提供了一个路径图：

| 阶段 | 典型年龄 | 核心动作 |
|------|---------|---------|
| 阶段一：大公司积累 | 30–55岁 | 最大化 401K、RSU、ESPP；建立应税账户 |
| 阶段二：自由职业过渡 | 55–62岁 | 降低工作强度，收入覆盖支出，让组合继续增长 |
| 阶段三：完全退休 | 62岁+ | 按 4% 规则提取；67岁开启社保 |

判断自己在哪个阶段，比单纯看"账户余额够不够"更有价值——它给出了下一步行动方向。

### 第四步：理解序列风险（Sequence of Returns Risk）

很多人以为"只要平均年化8%，就不会有问题"。这个逻辑在积累期成立，在提取期会失效。

[[wiki/退休规划/Sequence-of-Returns-Risk\|序列收益风险]] 指的是：退休后头几年如果市场大跌，叠加每年提取操作，会导致本金永久性损耗，即使后续市场反弹也难以恢复。

**为什么50岁需要开始关注这件事：**

退休前后各5年（大约55–65岁）是"红色警戒区"——资产规模最大、提取刚刚开始，开局熊市的破坏力最强。现在还有10年，正是建立缓冲机制的窗口。

常见应对手段：
- 保留1–3年生活支出的现金或短债（SGOV/T-Bills）
- 接近退休时逐步降低股票仓位（Bond Tent 策略）
- 退休初期保持灵活支出，市场下跌时主动削减非必要开销

### 第五步：评估 Roth Conversion 窗口

如果你有大量 Traditional 401K/IRA，而退休后的前几年收入会显著降低，这就是一个低税率窗口——把 Traditional 账户的钱转换为 Roth，未来提取免税，还能降低 RMD（72岁强制最低提取）压力。

[[wiki/税务策略/Roth-Conversion-Ladder\|Roth Conversion Ladder]] 的核心逻辑：在退休后低收入年份，每年将一定金额从 Traditional 转入 Roth，控制在较低税率区间（如 22% 档），每批转换的本金满5年后可免税提取。

对于50岁、还在高收入阶段的人，Roth Conversion 当前性价比不高（税率太高）。但这是值得规划的未来机会——退休后收入骤降，转换窗口随之打开。

## 阿明的分析

### 第一步：FI Number

阿明估算退休后年支出约 $120,000（加州生活，含医保、出行、偶尔旅行）。考虑到提取时需要缴纳联邦和州税，以税前支出来计算目标组合规模：

$$\text{FI Number} = \$120,000 \times 25 = \$3,000,000$$

### 第二步：当前进度

| 账户 | 余额 |
|------|------|
| 401K（传统） | $800,000 |
| Roth IRA | $120,000 |
| 应税账户 | $200,000 |
| **合计** | **$1,120,000** |

$$\text{进度} = \frac{\$1,120,000}{\$3,000,000} \approx 37\%$$

离目标还差 $1.88M，看起来遥远。但这忽略了复利的力量。

### 第三步：预计达标时间

假设组合年化增长 7%，每年继续存入约 $60K（最大化 401K + 应税账户追加）：

| 假设条件 | 预计达标年份 | 阿明当时年龄 |
|---------|------------|------------|
| 7% 年化，每年存 $60K | 约 2036 年 | 60岁 |
| 6% 年化，每年存 $60K | 约 2037 年 | 61岁 |
| 7% 年化，每年存 $40K | 约 2038 年 | 62岁 |

结论：**阿明大概在 60–62 岁可以达到 FI Number**，前提是不发生重大市场崩溃或支出大幅增加。

这个结论是乐观估计。阿明所在阶段（当前阶段一末尾）的核心任务不是"快速存钱"，而是：

1. **继续最大化税前储蓄**：2025 年 401K 上限 $23,500（50岁以上可追加 $7,500 catch-up = 共 $31,000）
2. **建立序列风险缓冲**：应税账户中保留1–2年支出的现金或短债，不要全仓股票
3. **规划 Roth Conversion 入口**：现在收入高，暂不转换；但需规划退休第1年起开始分批转换 Traditional 401K

### 当前行动清单

| 优先级 | 行动 | 时间节点 |
|-------|------|---------|
| 高 | 最大化 401K 含 catch-up（共 $31K/年） | 每年 |
| 高 | 应税账户保留 $120K–$180K 现金/短债作序列风险缓冲 | 58岁前完成 |
| 中 | 退休前与 CPA 规划 Roth Conversion 起点年份和金额 | 55–58岁 |
| 中 | 与财务顾问讨论 Bond Tent 降仓计划（60岁时股票比例从90%逐步降至60%） | 57岁开始 |
| 低 | 估算社保预期金额（ssa.gov 查询），规划延迟领取到 70 岁 | 60岁前 |

### 阿明的结论

"我在正确的路上，但不能停止思考。37% 的进度说明我不能提前退休，但 60–62 岁是现实目标。接下来10年不是拼命存钱，而是做好缓冲、规划税务。"

## 用 Claude 复现

上面的分析是用阿明的虚构数字演示的。如果你想对自己的情况运行类似分析，可以按以下步骤操作：

### 前置条件

- 安装 [Claude Code](https://claude.ai/code)（Anthropic 官方 CLI）
- 克隆本仓库

### 步骤

**1. 克隆仓库**

```bash
git clone https://github.com/austinxyz/wealth.git
cd wealth
```

**2. 填写你的财务档案**

```bash
cp templates/基本信息.md output/基本信息.md
cp templates/账户与资产.md output/账户与资产.md
cp templates/财务目标.md output/财务目标.md
# 用任意编辑器打开这些文件，填入你的真实数据
```

> `output/` 目录被 `.gitignore` 忽略，你的个人数据不会上传到 GitHub。

**3. 启动 Claude Code 并运行退休分析**

```bash
claude
```

在 Claude Code 的交互界面中输入：

```
/wealth-advise 退休规划
```

Claude 会读取你的 `output/` 档案和 `wiki/` 知识库，针对你的具体数字生成分析，包括：

- 你的 FI Number 和当前进度
- 预计达标年份（多种情景假设）
- 序列风险敞口评估
- Roth Conversion 时机建议
- 当前阶段的优先行动清单
