# wealth-advise

结合 wiki/ 通用知识和 output/ 个人财务档案，为指定主题生成针对性建议。

## 前置条件

使用前请确保已完成以下步骤：
1. 将 `templates/` 中的模板文件复制到 `output/` 并填写个人数据
2. 至少填写 `output/基本信息.md` 和 `output/账户与资产.md`

详见 `templates/README.md`。

## 使用方式

```
/wealth-advise 退休规划
/wealth-advise 账户开立
/wealth-advise 税务优化
/wealth-advise 中美双边资产
/wealth-advise 财务健康诊断
```

## 主题到文件的映射

| 主题关键词 | 相关 wiki 条目 | 相关 output 文件 |
|-----------|--------------|----------------|
| 账户开立 / 新手入门 | wiki/账户类型/* | output/基本信息.md, output/财务目标.md |
| 税务优化 / 税损收割 | wiki/税务策略/* | output/收入与税务.md, output/账户与资产.md |
| 退休规划 / FIRE | wiki/退休规划/* | output/账户与资产.md, output/财务目标.md |
| 投资配置 | wiki/投资品种/*, wiki/账户类型/* | output/账户与资产.md, output/财务目标.md |
| 中美双边 / 回国 | wiki/中美对比/* | output/基本信息.md, output/账户与资产.md |
| 保险规划 | wiki/保险规划/* | output/基本信息.md, output/收入与税务.md |
| 财务健康诊断 | wiki/家庭财务/*, wiki/退休规划/* | output/基本信息.md, output/收入与税务.md, output/账户与资产.md, output/财务目标.md |

## 执行步骤

1. **解析主题**：根据用户输入，对照上表确定需要读取的文件列表

2. **读取知识层**：使用 Read 工具读取相关 wiki 条目，提取适用的原则、规则、要点

3. **读取个人档案**：使用 Read 工具读取 output/ 中对应的模板文件。如果文件不存在，提示用户先按 `templates/README.md` 的说明填写

4. **生成分析**：综合知识层和个人数据，生成以下结构的建议：

```markdown
## 你的情况概述
基于你的档案，当前 <主题> 方面的状况是：...

## 发现的优化机会
1. <具体问题>：<基于 wiki 知识的解释>
2. ...

## 具体行动建议
| 优先级 | 操作 | 预期价值 | 注意事项 |
|-------|------|---------|---------|
| 🔴 高 | ... | ... | ... |

## 风险提示
- ...
```

5. **询问是否保存**：询问用户是否将建议保存到 output/ 目录
   - 是 → 写入 `output/<主题>策略建议-<日期>.md`
   - 否 → 仅展示，不写入文件

## 注意事项

- output/ 目录含个人敏感数据，已被 .gitignore 忽略，不会上传到 GitHub
- 本工具输出为参考建议，不构成投资建议，请结合自身判断并咨询专业顾问
- 如需更新建议，修改 output/ 中的档案文件后重新运行 /wealth-advise
```

# 来美第一年该开哪些账户

## 场景

小陈刚拿到第一份 offer，入职前看了一圈各种账户名词——401K、Roth IRA、HSA、HYSA——完全不知道从哪里开始，也不确定 H1B 身份能不能开这些账户。

## 怎么做

来美国工作，账户体系和国内完全不同。先建立一个优先级框架，按顺序来：

**第一优先：401K（获取公司匹配）**

401K 是通过雇主开的退休储蓄账户，税前缴纳，投资增长免税，退休后取款时缴税。最重要的一点：大多数公司提供匹配（match），比如"员工缴 6%，公司匹配 3%"。**不缴到获得完整匹配，就等于放弃了免费的工资。** H1B 可以参与 401K，无身份限制。

> 📚 深入阅读：[[wiki/账户类型/401K\|401K 完整解析]]

**第二优先：紧急基金**

在继续投资之前，先在高息储蓄账户（HYSA）存够 3-6 个月生活费。这是财务安全的底线，防止意外支出逼你提前取退休账户（提前取款罚 10%）。

> 📚 深入阅读：[[wiki/家庭财务/紧急基金\|紧急基金]]

**第三优先：Roth IRA**

Roth IRA 是个人开的退休账户（不通过雇主），税后缴纳，未来增长和取款**完全免税**。2025 年每人每年上限 $7,000。H1B 有合法工作收入即可开立。

收入限制：单身年收入超过 $161,000 后开始逐步失去直接缴纳资格，超过 $176,000 无法直接缴纳（但可通过 Backdoor Roth 解决，收入高时再考虑）。

> 📚 深入阅读：[[wiki/账户类型/Roth-IRA\|Roth IRA 完整解析]]

**第四优先：HSA（如果公司提供 HDHP 医保）**

如果公司的医保计划包含高免赔额计划（HDHP），可以开 HSA。HSA 是三重免税账户：存入免税、增长免税、医疗支出取款免税。是所有账户里税务效率最高的，值得优先填满。

> 📚 深入阅读：[[wiki/账户类型/HSA\|HSA 完整解析]]

## 小陈的选择

小陈入职后：

1. **立刻**：HR 系统里设置 401K 缴纳比例至少达到公司匹配上限（公司匹配 4%，他设置了 4%）
2. **第1个月**：开了一个 Marcus（Goldman Sachs）的 HYSA，工资自动转入，目标是存够 3 个月房租+生活费（约 $9,000）
3. **3个月后**：在 Fidelity 开了 Roth IRA，设置每月自动定投 $500（年末凑够 $6,000）
4. **公司医保选择**：对比了 PPO 和 HDHP，因为身体健康、用医少，选了 HDHP 并开了 HSA，每年存满 $4,300（2025 年单人上限）

优先级总结：**401K匹配 → 紧急基金 → Roth IRA → HSA → 继续增加 401K**

# 50岁如何评估退休准备度

## 场景

阿明今年 50 岁，在大公司做了快 20 年，退休账户积累了相当的金额。但他说不清楚自己"够了没有"——账户里的数字够大，但他不知道用什么标准来判断，也不清楚什么时候可以退休。

## 怎么做

评估退休准备度，核心是两个数字的比较：**你有多少** vs **你需要多少**。

**第一步：算出你的"财务独立数字"**

财务独立数字 = 退休后年支出 × 25

这来自于广为接受的 **4% 规则**：历史数据显示，退休组合每年提取 4%，在 30 年内有极高的概率不会耗尽。反过来说，如果你的资产是年支出的 25 倍，你就达到了财务独立。

例如，退休后每年需要 $120,000（含税），财务独立数字就是 $3,000,000。

> 📚 深入阅读：[[wiki/退休规划/4%规则\|4% 规则完整解析]]

**第二步：评估你现在在哪里**

把所有退休账户余额加起来（401K + IRA + 应税账户），与财务独立数字对比。50 岁时常用的参考基准：年收入的 7-10 倍视为"步入正轨"。

**第三步：考虑序列风险**

如果你退休后市场早期大跌，可能比预期更快耗尽资产——这叫 Sequence of Returns Risk（收益序列风险）。临近退休时，通常需要逐渐降低股票比例、增加债券缓冲，俗称"债券帐篷（Bond Tent）"。

> 📚 深入阅读：[[wiki/退休规划/Sequence-of-Returns-Risk\|序列风险]]

**第四步：规划 Roth Conversion 窗口**

很多高薪工作者在职时税率高，不适合做 Roth 转换。退休后、社保开始前（67 岁）的这段时间，往往是税率最低的"黄金窗口"——可以把传统 401K/IRA 的钱分批转换成 Roth，享受更低税率，同时降低未来的 RMD 压力。

> 📚 深入阅读：[[wiki/税务策略/Roth-Conversion-Ladder\|Roth Conversion 阶梯策略]]

## 阿明的分析

阿明做了以下评估：

**他有多少**：401K $800K + Roth IRA $120K + 应税账户 $200K = 合计约 $1,120,000

**他需要多少**：预计退休后年支出 $120,000，财务独立数字 = $3,000,000

**结论**：目前积累约为目标的 37%，距离完全财务独立还有一段距离。但他 50 岁，还有 10-15 年工作期，以当前储蓄率持续积累，有望在 60-62 岁前后达到目标。

**阿明的行动计划**：
1. 继续最大化 401K（2025 年 50 岁以上上限 $31,000，含追加缴纳）
2. 在 60 岁退休、67 岁领社保之间的 7 年里，系统性做 Roth Conversion，填满 22% 税率区间
3. 开始将应税账户逐渐调整为更保守的配置，建立 2-3 年的现金/短债缓冲

按三段式退休框架（大公司 → 自由职业 → 完全退休），阿明计划 55 岁后可能先过渡到兼职顾问，降低强度，同时继续积累。

> 📚 深入阅读：[[wiki/退休规划/三段式退休框架\|三段式退休框架]]

# 回国前必做的财务清单

## 场景

王女士已经拿到绿卡 12 年了，丈夫一直在国内，她自己也越来越确定会在 5 年内回国定居。她对财务清单毫无概念，只知道"回国前要处理一堆事"，但不知道从哪里开始，也不知道最坏的情况是什么。

## 怎么做

**Q：绿卡一定要放弃吗？**

不一定。但如果你持有绿卡超过 8 年，并且连续大部分时间住在美国，就属于"长期居民（Long-Term Resident, LTR）"。LTR 在放弃绿卡时需要面对与放弃公民身份相同的 Exit Tax 规则。因此，关键是**在满 8 年之前做出决定**，还是继续持有绿卡（意味着继续有申报义务和潜在税务负担）。

> 📚 深入阅读：[[wiki/中美对比/LTR-8年窗口\|LTR 8 年窗口]]

**Q：Exit Tax 是什么？会很贵吗？**

Exit Tax 是美国税法对放弃国籍或放弃绿卡（LTR）的人设计的"离境税"。简单来说，税务局会把你视为在离境当天"卖掉了所有资产"，对未实现收益征税。是否需要缴 Exit Tax，取决于你是否满足 Covered Expatriate 的三个测试（净资产、5 年均税、税务合规）之一。

> 📚 深入阅读：[[wiki/中美对比/Exit-Tax\|Exit Tax 详解]]

**Q：回国前我需要做哪些财务清理？**

大致清单如下（优先级从高到低）：

1. **确认绿卡策略**：是继续持有、申请放弃，还是转公民？各有利弊，需提前 1-2 年决定
2. **处理退休账户**：是否在离境前做 Roth Conversion？账户在美国可以继续持有，但需要考虑未来取款时的税务归属
3. **清理应税账户**：考虑在离境前实现部分收益（趁仍是美国税居民），或根据退出策略保留
4. **整理中国资产**：房产、存款、股票是否需要在离境前处置？中国外汇管制对转出有限制
5. **申报合规检查**：确认 FBAR 和 FATCA 历史申报记录完整，无遗漏年份
6. **Medicare 处理**：如果已开始缴纳 Medicare，考虑回国后如何处理（海外无法使用）

> 📚 深入阅读：[[wiki/中美对比/回国前资产清理\|回国前资产清理完整指南]]

**Q：FBAR 是什么？我必须申报吗？**

FBAR（Foreign Bank Account Report）要求所有美国税务居民，如果在一年中任意时间，所有境外金融账户余额合计超过 $10,000，就必须每年申报。申报截止日一般是 4 月 15 日（自动延期至 10 月）。未申报的罚款极重，最高可达账户余额的 50%。

> 📚 深入阅读：[[wiki/中美对比/FBAR\|FBAR 完整解析]]

## 王女士的处境

王女士持有绿卡 12 年，已经是 LTR，Exit Tax 问题无法回避。她的主要资产（中美合计折算约 $150 万）也接近 Covered Expatriate 的净资产门槛。

她做了以下决定：

2. 提前 3 年开始系统性 Roth Conversion，降低传统账户余额（减少 Exit Tax 基数）
3. 中国房产计划在回国后再处置（中国境内交易，不在美国 Exit Tax 范围内）
4. 聘请 CPA 做 3 年内 FBAR/FATCA 的回溯审查，确保合规
5. Medicare Part A 继续保留（已缴满年限，免月费），放弃 Part B（月费无意义）

> 📚 深入阅读：[[wiki/中美对比/Medicare海外策略\|Medicare 海外策略]]
