## 定义

**Automated Savings**：通过定期自动转账 / 雇主直接扣款，将储蓄从"月底剩下就存"变成"月初强制扣除"。核心哲学：**系统胜过自律**，人性会在繁忙或情绪化时偷懒，自动化不会。

## 核心要点

**核心原则**：**人性 vs 系统——选系统**。
- 手动每月转账 → 容易跳过（"这月特殊"）
- 自动转账 → 强制执行，无需决策疲劳
- 行为金融：默认选项的力量（Opt-out > Opt-in）

**典型自动化流水线**（每月 1-5 日触发）：

```
工资到账
  ↓
[雇主直接扣款]
  → 401K（拉满 $24,500/年 + match）
  → HSA（拉满 $8,750 家庭）
  → ESPP（最多 15% 工资）
  ↓
[剩余进 Checking]
  ↓
[自动转账至 HYSA / 投资账户]
  → Roth IRA（$7,500/年 ÷ 12 = $625/月，Backdoor 形式）
  → 529 子女教育（$500/月）
  → 紧急基金补充（直到目标）
  → Taxable 投资（Betterment / Fidelity Go 自动投）
  ↓
[自动扣款固定支出]
  → 房贷、车贷、水电、保险
  ↓
[剩余才进"消费账户"]
  → 食品、娱乐、购物
```

**四大自动化工具**：

### 1. 雇主端扣款（最优先）
- **401K Pre-tax + Roth**：直接从工资扣，永不过手
- **HSA**：通过 Payroll 扣（省 FICA 税 7.65%，比个人存 HSA 多省）
- **ESPP**：15% 工资自动购买折扣股票
- **团体保险保费**：Pre-tax 扣除

### 2. 银行自动转账（ACH Auto-Transfer）
- 主 Checking → HYSA（紧急基金、短期目标）
- 主 Checking → Roth IRA（$625/月）
- 主 Checking → 529（$500/月）
- 设置在**发工资后 1-3 天**（钱到就转走）

### 3. 投资账户自动投资
- **Betterment / Wealthfront**：自动 Robo 配置 + 再平衡 + 税损收割
- **Vanguard / Fidelity**：Automatic Investment Plan，每月自动买 VTI/VOO
- **Dividend Reinvestment（DRIP）**：分红自动再投
- **[[Mega-Backdoor-Roth]]**：After-tax 401K 自动供款 + In-Plan Roth Conversion

### 4. 加薪自动升级（Save More Tomorrow）
- **加薪时**自动提升 401K 供款比例（如 +50% 加薪进储蓄）
- 避免生活方式通胀（Lifestyle Inflation）
- 心理上"未拿到过的钱不心疼"

**自动化的心理学基础**：
- **默认选项效应**：401K 默认 opt-in（美国公司必须）显著提高参保率
- **心理账户**（Richard Thaler）：钱分账户后不易混用
- **现状偏见**：设置好后很少改动
- **Pay Yourself First**：哲学的执行层

**自动化 ≠ 不关心**：
- 每季度复盘一次（金额是否够、比例是否对）
- 加薪 / 换工作 / 生子 / 换房后调整
- 紧急基金**满了**后把自动转账金额改投到 Taxable（否则堆积浪费）

**自动化陷阱**：
- ❌ **Checking 余额不足**导致自动扣款失败（透支费 + 信用影响）→ 留 1 个月支出缓冲
- ❌ **加薪后不调整**→ 储蓄比例下降
- ❌ **紧急支出时硬撑自动扣款**→ 刷信用卡还，得不偿失；应主动暂停
- ❌ **设置一次忘记跟踪**→ 年度复盘必做

**推荐自动化设置清单**：

| 目标 | 金额 | 账户 | 频率 |
|------|------|------|------|
| 401K Pre-tax | $24,500/年 | Fidelity/Vanguard 计划 | 每发薪周期 |
| 401K Roth | 按需 | 同上 | 同上 |
| HSA（HDHP 家庭 2026） | $8,750/年 | Payroll 或 Fidelity | 同上 |
| [[Mega-Backdoor-Roth]]（若计划支持） | 最多 $47,500 | After-tax → Roth | 同上 |
| Roth IRA（Backdoor） | $7,500/年 | Fidelity/Vanguard | $625/月或年初一次 |
| 529 | $500/月 | Utah/NY 529 | 月初 |
| [[紧急基金]]补充 | 按缺口 | HYSA | 月初 |
| 额外 Taxable | 剩余 | Betterment/Fidelity | 月初 |

**加薪前自动化建议**（让储蓄跑在收入前）：
```
- 加薪 +$10k/年
  → $5k 进 401K / HSA / Roth（储蓄率 +50%）
  → $3k 进 Taxable
  → $2k 生活升级（感到"加了薪"）
```
