## 定义

**Emergency Fund（紧急基金）= 专用于突发事件的高流动性现金储备**，目的是**防止被迫卖出投资或借债**以应对意外开销。是财务规划的**"基石"**——没有紧急基金，其他策略都脆弱。

## 核心要点

**典型紧急场景**：失业 / 收入中断、大病或受伤（医疗自付）、房屋大修（屋顶、HVAC、水管）、车辆事故 / 大修、家人紧急医疗、宠物急诊、意外旅行（亲人离世）。

**不使用投资组合应急的原因**：
- 市场下跌时卖出 → 锁定亏损
- 卖出触发资本利得税
- 退休账户提前取 → 10% 罚款 + 税
- Roth IRA 本金可取但损失**免税增长空间**
- 卖出时机往往最糟（熊市 = 失业潮）

**目标规模：3-6 个月"必要支出"（非总收入）**：

| 情况 | 建议月数 |
|------|---------|
| 单身 + 稳定工作 + 低开销 | **3 个月** |
| 双收入家庭 + 稳定行业 | 3-4 个月 |
| 双收入 + 不稳定行业 | 6 个月 |
| 单收入家庭 | **6 个月** |
| 自雇 / 合同工 / 销售（佣金） | 6-12 个月 |
| 高科技行业（裁员风险） | 6-9 个月 |
| 预计退休或 FIRE | **12-24 个月**（现金缓冲） |
| 有大额未知支出（医疗、诉讼） | 12 个月+ |

**"必要支出"的边界**：

| 纳入 | 不纳入 |
|------|--------|
| 房租 / 房贷 | 外食 / 娱乐 |
| 食品、日用 | 旅行 |
| 水电气、网络 | 订阅服务 |
| 基本保险（健保、车险） | 额外投资 |
| 必要通勤 | 奢侈开销 |
| 最低还款（信用卡、车贷、学贷） | |
| 基本儿童照护 | |

**计算方法**：
```
第 1 步：月度必要支出 = $X
第 2 步：X × 目标月数 = 紧急基金目标
第 3 步：加 1-2 月心理 buffer（焦虑型可加）
```

**加州双职工示例**：
- 月必要支出 $10,000 × 6 个月 = $60,000 + buffer = **$70,000**

**放置原则**（三要素，缺一不可）：
1. **本金保全**：FDIC 保险或美国国债
2. **高流动性**：1-3 个工作日可取
3. **收益率**：至少跟上通胀

**不适合的容器**：股票 / ETF（本金风险）、长期债券（利率风险）、加密货币（波动极大）、房产（不流动）、IRA / 401K（罚款 + 税）。

**建立顺序（Financial Priority）**：
```
1. $1,000 启动基金（应对小事故）
2. 拉满 401K 匹配（免费钱不能放弃）
3. 高息债务清除（信用卡 15%+ 利率）
4. 3-6 月紧急基金（主战场）
5. 之后：退休储蓄、买房首付、教育基金
```

> ⚠️ **"先拉满 401K 再建紧急基金"不对** —— 失业时没现金会被迫取 401K，10% 罚款 + 普通所得税 = 总损失 30-40%。

**退休前 vs 退休后的角色变化**：
- **退休前**：应对失业的桥梁
- **退休后**：应对 [[Sequence-of-Returns-Risk]] 的缓冲（熊市时不动组合）
- 退休后紧急基金常合并进 [[Bucket-Strategy]] 的 Bucket 1（1-2 年支出）

**FIRE / 早退休者**：紧急基金 + Bucket 1 合并考虑，总现金缓冲 **2-5 年**（覆盖首 5 年 SRR 高风险期）。

**管理纪律**：
- **每季度/半年复盘**：支出变化？目标月数仍合适？利率仍有竞争力？
- **用完即补**：重建列为优先事项（**高于投资**）
- **避免过度储蓄**：> 12 个月（非退休期）机会成本大，多余转 ETF
- **与 [[现金流预算]] 联动**：必要支出数字要与预算类目口径一致

**不建议的"替代方案"**：
| 方案 | 问题 |
|------|------|
| HSA 付医疗 | HSA 是长期工具（终极 Roth），医疗应用紧急基金 |
| HELOC（Home Equity） | 批准慢、房价跌时被冻结；只能做**补充**非替代 |
| 加密货币 | 波动大，紧急时可能正暴跌 |
| 信用卡 | 利率 20%+，只可作极短过渡 |

**加州居民最佳容器**：[[SGOV]]（利息免加州税 9.3-13.3%），税后收益优于 HYSA。
