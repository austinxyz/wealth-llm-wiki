## 定义

**Pay Yourself First (PYF)**：工资到账后**第一件事**转储蓄/投资，**剩余**才是生活费。源自 George Clason《The Richest Man in Babylon》（1926），是几乎所有现代理财方法的共同哲学基石。核心反转："先消费后储蓄" → "先储蓄后消费"。

## 核心要点

**核心反转**：

| 传统做法（失败） | Pay Yourself First（有效） |
|----------------|-------------------------|
| 收入 - 支出 = 储蓄 | **收入 - 储蓄 = 支出** |
| 月底剩下就存 | 月初先存完 |
| 消费随心 | 消费受限（但刚够用） |
| 储蓄率 5-10% | 储蓄率 20-50%+ |

**为什么有效（行为金融依据）**：
1. **Parkinson 定律**：支出会扩张到填满所有可用收入，必须提前抽走储蓄
2. **心理账户**（Mental Accounting）：看不到的钱不会花
3. **现状偏见**：设置后很少改动
4. **损失厌恶**：取回"已储蓄"比"未储蓄"心理成本更高

**PYF 的"自己"层次**（从基础到高阶）：

```
第 1 层：还债的自己
  → 先付最低还款 + 超额还高息债务

第 2 层：应急的自己
  → 紧急基金 3-6 个月支出

第 3 层：退休的自己
  → 401K 到 match / Roth IRA 拉满 / HSA 拉满

第 4 层：富裕的自己
  → Backdoor / Mega Backdoor / Taxable 投资

第 5 层：代际的自己
  → 529、信托、遗产规划
```

**执行方法（必配 [[自动化储蓄]]）**：

```
工资到账日 +1 天：
  自动扣 401K（雇主端）
  自动扣 HSA（雇主端）
  自动转 Roth IRA $625（银行 ACH）
  自动转 紧急基金 / HYSA
  自动转 Taxable 投资账户
  ↓
剩余资金才进消费账户
```

**关键金额次序**（美国 2026 年优先级）：
1. 401K 到 **employer match**（白捡 100% 回报）
2. **高息债务还清**（> 7% 利率）
3. **紧急基金** 1 个月 → 3 月 → 6 月
4. [[HSA]] 拉满（$8,750 家庭，三重税优）
5. [[Roth-IRA]] 或 [[Backdoor-Roth-IRA]] 拉满（$7,500）
6. 401K 拉满至联邦上限（$24,500）
7. [[Mega-Backdoor-Roth]]（若计划支持，最多 $47,500）
8. 529 子女教育
9. Taxable Brokerage
10. 超额还房贷 / 房产投资 / 其他

**储蓄率与 PYF 效果**：

| PYF 储蓄率 | 退休时间（FIRE） |
|----------|----------------|
| 10% | 50+ 年 |
| 20% | 30 年 |
| 30% | 22 年 |
| 50% | 15 年（Regular FIRE） |
| 70% | 7-10 年（Lean FIRE） |

**常见误区**：

- ❌ **"等我加薪再开始 PYF"** → 加薪会被新支出吞掉（Lifestyle Inflation）
- ❌ **"先还完债再 PYF"** → 除非超高息，否则至少要同时做（错过 match + 复利）
- ❌ **"大额不规则收入不用 PYF"** → 奖金应按比例 PYF（如 50% 进储蓄）
- ❌ **"储蓄放活期够了"** → PYF 的钱要立即进投资/HYSA 防通胀

**加薪时的 PYF 升级**：

```
加薪 $10k/年（税后约 $7k）
  保守法：+$7k 全进储蓄 → 储蓄率 +100%
  平衡法：+$5k 储蓄 + $2k 消费升级 → 储蓄率 +70%
  放纵法：+$0 储蓄 → 错过复利机会
```

**PYF vs 其他预算方法的关系**：

| 预算方法 | PYF 体现 |
|---------|---------|
| [[50-30-20规则]] | Savings 20% 必须 PYF |
| [[零基预算]] | 第一类分配必须是 Savings |
| [[信封预算法]] | 先装 Savings 信封 |
| 无预算 + PYF | 也能工作（只要自动化 Savings） |

**PYF 的历史演进**：
- 1926：George Clason 在《The Richest Man in Babylon》提出 "Start thy purse to fattening"（先让钱包增重）
- 1994：T. Harv Eker 《Secrets of the Millionaire Mind》推广
- 2000s：Dave Ramsey 融入 Baby Steps
- 2010s：FIRE 社区推向极致（50-70% 储蓄率）
