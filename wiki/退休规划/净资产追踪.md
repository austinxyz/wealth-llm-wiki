## 定义

**净资产（Net Worth）= 总资产 - 总负债**。这是衡量个人财务健康的单一最重要指标：工资是"流量"，净资产是"存量"，也是判断财务独立进度的标准。追踪频率建议至少每季度，理想是每月。

## 核心要点

**资产构成四大类**：

| 类别 | 主要科目 |
|------|---------|
| 流动性（Liquid） | Checking / Savings / HYSA / MMF / [[SGOV]] / T-Bills |
| 投资（Investment） | Taxable Brokerage / [[401K]] / [[Traditional-IRA]] / [[Roth-IRA]] / [[HSA]] / [[529-Plan]] |
| 不动产（Real Estate） | 自住房（Zillow/Redfin）/ 投资房 / 土地 |
| 其他（Other） | 车辆（KBB）/ 珠宝艺术品 / 企业股权 / 加密货币 / 应收账款 |

**负债构成**：房贷 / HELOC / 投资房房贷 / 车贷 / 学贷 / 信用卡余额（未还清）/ 医疗债务 / 个人贷款 / 税务欠款。

**战略意义**：
- **FIRE 进度**：FIRE 进度 = 当前净资产 / FIRE 数字 × 100%（FIRE 数字 = 年支出 × 25）
- **资产配置分析**：股票/债券比例、应税 vs 税延 vs 免税 比例、集中度风险
- **债务/资产比例**：< 30% 健康；30-50% 正常（有房贷）；> 50% 警戒
- **跨年比较**：年度净资产增长 = 储蓄 + 投资回报 + 房产升值
- **心理作用**：看见长期曲线向上 → 熊市不恐慌卖出

**追踪的六个常见错误**：
1. **不追踪**（"大概知道"≠ 实际准确）
2. **过度粒度**（每天更新 → 焦虑；每月一次够了）
3. **只看资产，不看负债**
4. **夸大房产价值**（按 Zillow 最高估 → 不真实）
5. **忘记税务折扣**（Traditional 401K $500k ≠ Roth IRA $500k）
6. **忽略小资产**（[[HSA]]、[[529-Plan]]、已归属 [[RSU]]、加密货币）

**历史数据保留**：
- 每月快照 + 年度汇总
- 备份到云（Google Drive / iCloud）
- 敏感数据加密

**简单 Spreadsheet 模板结构**：
```
| 月份 | 总资产 | 总负债 | 净资产 | 环比变化 |
| 2026-01 | $1,450k | $405k | $1,045k | +2.3% |
| 2026-02 | $1,485k | $404k | $1,081k | +3.4% |
```
