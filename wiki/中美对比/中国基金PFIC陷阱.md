## 定义

**中国基金 PFIC 陷阱**：美国人持有中国公募基金 / ETF / 理财 / 储蓄险 / 余额宝等产品自动触发 [[PFIC]] 惩罚性税务规则 —— 有效税率 **50-80%** + 每只基金每年 **Form 8621** 合规 + 税务追诉期**无限**。这是跨境华裔家庭**最常踩的单一最大税务地雷**。本条目列出具体产品识别清单 + 实务清理策略。

## 核心要点

### 哪些中国产品是 PFIC（几乎全部）

**✅ 100% 是 PFIC**：

| 产品类型 | 典型例子 |
|---------|---------|
| **公募基金（股票型）**| 易方达蓝筹、兴全合宜、中欧医疗健康 |
| **公募基金（债券型）**| 易方达信用债、富国天利、中银产业债 |
| **公募基金（混合型）**| 兴全合润、富国天惠、中欧养老 |
| **指数基金 / ETF** | 沪深 300ETF、上证 50ETF、创业板 ETF |
| **货币基金** | 余额宝（天弘）、零钱通、理财通、活期宝 |
| **QDII 基金**（投海外）| 易方达全球蓝筹、兴全全球视野 |
| **FOF 基金** | 交银瑞思、中欧预见养老 |
| **香港互认基金** | 富邦香港、汇丰环球投资 |
| **银行理财（结构性）**| 招行朝招金、建行乾元 |
| **银行理财（固收+）**| 工行鑫得利、建行利得盈 |
| **保险储蓄型**（分红/万能/投连）| 平安福、太保金佑、新华成长相伴 |
| **信托计划** | 信托投资公司产品 |
| **私募基金** | 如投资在美国境外设立 |

### ❌ 不是 PFIC（安全）

| 产品 | 说明 |
|------|------|
| **中国 A 股个股** | 股票本身是运营公司，非 PFIC |
| **港股个股** | 同上（腾讯 700、阿里 9988）|
| **香港 ADR（美国券商）** | 如 BABA、JD、NIO 在美国券商 |
| **纯消费型保险** | 医疗险、重疾险、意外险（仅理赔无储蓄）|
| **实物黄金** | 非金融产品 |
| **房产** | 非金融，但有其他税务规则 |
| **中国银行存款** | 纯储蓄，非投资公司 |

### 隐蔽的 PFIC 陷阱

**⚠️ 容易被忽略的 PFIC**：

#### 1. 余额宝 / 零钱通 / 理财通
- 用户认为是"活期钱包"
- 实际是**货币基金**（天弘余额宝基金）
- **是 PFIC**
- 超 $10k → FBAR + Form 8938 + Form 8621

#### 2. 银行理财（非保本浮动收益）
- 招行朝招金 / 工行鑫得利等
- 实际是**集合理财产品**
- 2020+ 新规后更接近基金性质
- **是 PFIC**

#### 3. 储蓄型保险（分红/万能/投连）
- "儿童教育金"、"年金险"、"万能险"
- 宣传中强调"保障 + 理财"
- 有现金价值 + 分红
- **是 PFIC**（保险公司的投资账户被穿透）

#### 4. 第三方理财平台的"月月赚"、"活期宝"
- 支付宝、京东金融、陆金所
- 包装成"类活期"或"7 天理财"
- 底层仍是基金
- **是 PFIC**

#### 5. 企业年金 / 职业年金
- 中国企业补充养老
- 类似美国 401K 但**不享受美国税延**
- 可能被视为 PFIC（复杂）
- 至少要 [[Form-8938]]

#### 6. 国债逆回购
- 通过交易所做的短期投资
- 若账户内部分配到基金 → 可能 PFIC
- 单笔国债逆回购通常非 PFIC

### 场景化陷阱

**场景 1：回国探亲买"理财体验"**
```
父亲去招行帮你"存定期更好利率"
实际买了 6 个月理财产品（R2 稳健型）
金额 ¥500,000 ≈ $73,000
→ 自动触发 PFIC
→ 每年 1 份 Form 8621
→ 卖出时 Excess Distribution 规则
→ 有效税率 40-60%
→ 原本好意的"存定期"变成税务噩梦
```

**场景 2：支付宝"余额自动理财"开关**
```
支付宝默认"余额宝"开关打开
每月工资进入后自动转入余额宝（货币基金）
余额 $10k+
→ FBAR + Form 8938（若合计超门槛）+ Form 8621
→ 每年合规 $200-800 per fund
```

**场景 3：孩子"教育金保险"**
```
父母买了"XX 儿童教育金"保险 20 年
年交 ¥50,000 × 20 年 = ¥1,000,000
有现金价值 + 分红
→ 是 PFIC（保险储蓄部分）
→ 每年 Form 8621
→ 终止时 Excess Distribution 爆雷
```

**场景 4：QDII 基金"海外配置"**
```
中行推荐"易方达全球蓝筹"
说"投资美股降低波动"
实际是 QDII 基金（中国发行 + 投美股）
→ 仍是 PFIC（发行主体是中国基金公司）
→ 比买美国 VOO 多 50-80% 税负
```

### 有效税率对比

**同样 $100k 投资 5 年，增值 50%（利得 $50k）**：

| 投资工具 | 5 年后税负 | 有效税率 |
|---------|---------|---------|
| **美国 VTI**（长期资本利得）| $7,500-$10,000 | 15-20% |
| **美国 VTI**（0% LTCG 档）| $0 | 0% |
| **中国公募基金（MTM）**| $15,000-$18,500 | 30-37% |
| **中国公募基金（Default）**| **$30,000-$40,000** | **60-80%** |

**10 年对比**：
```
投 $100k → 10 年后 $200k（7% 年化）

美国 VTI 卖出：
  税 15% × $100k 利得 = $15,000
  净 $185,000

中国基金 Default 卖出：
  Excess Distribution 分摊 10 年
  每年 $10k × 37% + 利息罚款 ~5%/年复利
  合计税 + 利息罚款 ≈ $80,000
  净 $120,000

差距：$65,000（33%）
```

### 清理策略时间表

**发现 PFIC 后的行动顺序**：

```
立即（第 1 周）：
  □ 停止所有新增购买（包括自动定投、余额自动理财）
  □ 关闭支付宝 / 微信 "自动购买理财" 开关
  □ 记录当前所有 PFIC 的持仓 + 买入成本

1 月内：
  □ 咨询跨境 CPA（PFIC 专长）
  □ 决定清理顺序（按税务影响优先级）
  □ 评估是否一次性卖出 vs 分年卖出

3-6 月内：
  □ 卖出清理（最好在低收入年）
  □ 关闭空账户
  □ 补报历史 Form 8621（若持有多年）

年度：
  □ 每年持有期间必须 Form 8621 per fund
  □ 卖出当年按 Default / MTM 申报
```

**清理的税务含义**：

**Default（未做选择）清理**：
- 假设持有 3 年，现在卖出
- 利得 $50k → Excess Distribution
- 分摊到 3 年，每年 $16,667 按 37% → $6,167
- 加利息罚款（~8%/年复利）→ 约 $3,000
- 合计 ~$19,000-22,000 税
- 即 **40-45% 有效税率**

**MTM Election** 更好：
- 每年按增值部分普通税率
- 无 Excess Distribution 惩罚
- 但无法改回 Default（一旦选了就锁定）

**建议**：发现 PFIC → 请跨境 CPA 评估 → **当年做 MTM Election 减轻压力** → **下年或 2-3 年内清理完毕**

### 补报 Form 8621（漏报情况）

**过去年份漏报 Form 8621**：
- 税务追诉期**无限延长**至申报为止
- 即使 FBAR / 8938 都报了，8621 漏了 → 1040 仍未合规
- 补报路径：[[FBAR补报程序]] Streamlined（覆盖 8621 + FBAR + 8938 统一修补）

### 永久脱离 PFIC 的方法

**最根本：不买中国基金**
- 想要中国敞口？用美国 ETF：
  - **MCHI**（iShares MSCI China）—— 大盘
  - **KWEB**（KraneShares China Internet）—— 互联网
  - **FXI**（iShares China Large-Cap）—— 大蓝筹
  - **ASHR**（Xtrackers CSI 300）—— A 股
- 美国 ETF 虽投中国股票，**本身是美国基金**，**非 PFIC**

**保留中国敞口但避 PFIC**：
- 买**中国个股**（A 股 / 港股）
- 买**ADR**（美国上市的中概股）：BABA、JD、PDD、NIO
- 避免所有"基金"包装产品

### 与 [[回国前资产清理]] 的关系

**回国前是最后的清理窗口**：
- 仍是美国人 + 低收入年 → 最优卖出时机
- 回国后仍需 Form 8621（若保留绿卡）
- 回国后卖出可能触发中国税（若是中国税务居民 + 第 7 年后）
- 建议：**回国前 1-2 年全部清理**

### 对 [[Covered-Expatriate-三测试]] Test 3 的影响

**漏报 Form 8621 是 Test 3 最常见失败原因**：
- 即使净资产 < $2M 和税额 < $206k
- 漏报 Form 8621 → Test 3 失败 → **自动 Covered Expatriate**
- Exit Tax + Section 2801 全面适用
- 教训：放弃身份前务必用 Streamlined 补齐
