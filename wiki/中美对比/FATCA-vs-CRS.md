## 定义

**FATCA（美国版）** 和 **CRS（Common Reporting Standard，OECD 版）** 是全球两大金融信息自动交换机制。FATCA 是美国**单方面** 2010 年推出，CRS 是 OECD 2014 年推出的**多边**标准。**关键不对等**：美国**不参与 CRS**（只接收 FATCA 数据），造成美国成为外国人的"税务天堂"，而中国居民在美国的账户对中国税务是"黑盒"。

## 核心要点

**两大制度对比**：

| 维度 | FATCA | CRS |
|------|-------|-----|
| **发起方** | 美国单方 | OECD 多边 |
| **立法年** | 2010（HIRE Act）| 2014（CRS 发布）|
| **目标对象** | 美国税务居民 | 所有参与国居民 |
| **美国参与** | ✅ **是**（接收方）| ❌ **不参与** |
| **中国参与** | 事实执行（Model 1）| ✅ 2017 起正式 |
| **参与国数** | 113+（IGA） | 100+（CRS） |
| **报告频率** | 年度 | 年度 |
| **门槛** | $50k+（8938）| 无下限（逐账户）|
| **双向性** | IRS ↔ 部分 FFI 所在国 | **全球多边互换** |

### FATCA（美国单边）

- **信息流**：FFI → 本国政府 → IRS
- **惩罚机制**：不合规 FFI 接收美国款项预扣 30%
- **对象**：识别美国税务居民（公民 / 绿卡 / SPT）
- **表格**：[[Form-8938]]（个人端）+ FFI 报告（机构端）
- **互惠**：理论上 IRS 也提供部分美国账户数据给 IGA 对方，**但实际对等极不完整**

### CRS（OECD 多边）

- **信息流**：FFI → 本国政府 → 各参与国税务局（交叉匹配）
- **惩罚机制**：经济合作 + 软约束（非市场准入硬惩罚）
- **对象**：所有参与国的税务居民
- **中国执行**：2017 起，大陆居民在参与国账户数据**会被中国税局知道**
- **表格**：各国本地申报（中国：纳税人自我申报 + CRS 数据比对）

**关键不对等：美国不参与 CRS**：

```
外国 → 美国：
  外国人（中国居民）在美国开账户 → 美国不自动报给中国
  → 中国不自动知道
  → 美国成为"外国人税务天堂"

美国 → 外国：
  美国人在外国开账户 → FATCA 报给美国
  → IRS 自动知道
  → 无"秘密"可言
```

**对中美跨境家庭的具体影响**：

### 场景 1：美国人在中国的账户（FATCA 规制）
- 中国银行每年报 IRS
- 必须 FBAR + [[Form-8938]] 自报
- 完全透明

### 场景 2：中国居民在美国的账户（无机制）
- 美国不参与 CRS
- 中国税局 **默认不知道**
- 但中国 2024+ 加强海外资产审查
- 银行 KYC 仍会问"税务居民国"
- 不如实答：未来被发现仍有后果

### 场景 3：双重居民（美国 + 中国）
- **FATCA 方向**：美国人身份 → IRS 收到
- **CRS 方向**：CRS 看"哪国税务居民"
  - 若你填表报中国是税务居民 → CRS 会把美国账户数据报中国
  - 若你填表报美国 → 美国不是 CRS 参与国 → 不向中国报
- **实操**：填 W-9 + Form 8833 Tie-Breaker 是复杂决策

**中国 CRS 执行现状（2024-2026）**：
- 中国居民在港澳、新加坡、欧洲账户 → **中国税局已知**
- 中国对海外所得审查加强（"千万以上高净值"优先）
- 中国税务居民必须申报海外账户（类似美国 FBAR）
- 违规罚款：0.5-5 倍税款 + 滞纳金

**"美国税务天堂"策略（灰色地带）**：

某些中国富人利用美国不参与 CRS：
1. 开美国 LLC / Delaware Corporation
2. 资金流入美国券商 / 银行
3. **美国不自动报中国**
4. 中国端若未自行申报 → 可能长期未被发现

**但此策略正在失效**：
- 美国 2025+ 更多 IGA 对等条款
- Corporate Transparency Act（CTA）要求披露最终受益人
- 银行 KYC 加强（问税务居民身份）
- 中国主动征管 + 反避税规则（特别纳税调整）

**双边家庭的合规义务总结**：

| 你的身份 | FATCA（美国端）| CRS（中国端）|
|---------|--------------|------------|
| US Person + 在中国账户 | ✅ 美国人自报 + 中国 FFI 报 IRS | N/A（CRS 不针对美国人）|
| US Person + 在美国账户 | ✅ 美国常规申报 | N/A |
| 中国税务居民 + 在美国账户 | N/A（非 FATCA 目标）| **中国需自报**（CRS 不覆盖美国账户）|
| 中国税务居民 + 在非美海外账户 | N/A | ✅ **CRS 自动报中国** |

**回国华裔的 FATCA + CRS 双重义务（退休回国）**：

若保留绿卡回国：
```
美国账户：
  - FATCA 不适用（不是外国账户）
  - 美国 1099 → IRS
  - 中国端：若你是中国税务居民（[[中国6年规则]]第 7 年触发）→ 中国要求自报
  - CRS：美国不参与 → 中国不自动收到 → 但中国可能通过 [[中美税收协定]] 第 26 条（信息交换）请求

中国账户：
  - FATCA 适用：中国 FFI 报 IRS
  - [[FBAR]] + [[Form-8938]] 必报
  - 中国端：中国税务居民本地账户，无 CRS 自报义务（CRS 只报外国账户）
```

**自动交换 vs 按请求交换**：
- **FATCA / CRS**：自动每年交换
- **中美协定 Article 26**：按请求交换（2013 起执行）
- 即使美国不参与 CRS，中国仍可通过 Article 26 向 IRS 请求特定美国账户信息
- 实务：请求门槛较高，仅针对可疑重大案件

**关键实务建议**：

1. **US Person 不能利用"美国不参与 CRS"逃中国税**
   - 你是美国税务居民 → FATCA 仍然报
   - 即使 CRS 不覆盖，美国 FBAR + 8938 强制自报

2. **中国税务居民能否利用？**
   - 技术上美国账户不自动报中国
   - 但中国严格禁止（违反中国申报义务）
   - 2024+ 通过银行 KYC、外汇监测、主动调查发现率提升
   - 不推荐

3. **双重居民要格外小心**
   - W-9（美国）和 W-8（非美国）填哪个？
   - Tie-Breaker 决定后需一致
   - FATCA + CRS 可能同时触发报告

**CRS 的中国特殊规则**：
- 中国《非居民金融账户涉税信息尽职调查管理办法》（2017）
- 金融机构每年 3/31 前向税务局报告
- 覆盖：存款、证券、保险现金价值、基金、信托
- **门槛**：个人账户 $1M+ 高净值优先，低于仍报

**FATCA 豁免 vs CRS 豁免**：

| 账户类型 | FATCA 豁免 | CRS 豁免 |
|---------|----------|---------|
| 退休账户（401K/IRA）| ✅ | ✅（大多）|
| 合格保险 | ✅ | 部分 |
| 存款账户（< $50k）| 低门槛 | 部分机构 |
| HSA / 529 | ✅ | ✅ |
| 美国券商账户 | N/A | N/A（CRS 不覆盖）|
