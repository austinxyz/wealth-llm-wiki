## 定义

美国税法下有**四种身份**（公民、绿卡、税务居民、非居民），税务义务差异巨大。核心认知：美国是**全球仅有两国之一**按"公民身份"征税（另一是厄立特里亚）——意味着美国公民 / 绿卡**无论住哪里**都需申报全球所得。这是跨境资产规划**最根本的决策点**。

## 核心要点

**四种身份一览**：

| 身份 | 全球所得 | FBAR/8938 | 取得方式 | 脱离方式 |
|------|---------|----------|---------|---------|
| **US Citizen**（公民）| ✅ | ✅ | 出生 / 归化 | 放弃国籍 + [[Exit-Tax]] |
| **Green Card / LPR**（绿卡）| ✅ | ✅ | I-551 签发 | I-407 + 潜在 Exit Tax |
| **Resident Alien**（非绿卡税务居民）| ✅ | ✅ | [[Substantial-Presence-Test]] | 当年 < 31 天或 < 183 天 |
| **Nonresident Alien**（非居民）| ❌ 仅美国源 | ❌ | 未触发上述 | 保持状态 |

**Citizenship-Based Taxation（公民身份课税）**：
- 美国 + 厄立特里亚 = 全球仅此二国
- 其他国家（含中国）均按"居住地"课税
- 意味着：美国护照 → **一辈子全球申报**，直到放弃国籍
- 绿卡同样全球申报，直到 I-407

### 1. US Citizen（公民）

**税务义务**：
- 全球所得申报（1040）
- [[FBAR]]（海外账户 > $10k）
- Form 8938（FATCA，资产超门槛）
- [[PFIC]] Form 8621（每个中国基金每年）
- 即使 **全年 0 天在美国** 仍需申报

**可用减税工具**：

| 工具 | 2026 额度 | 适用 |
|------|---------|------|
| **Foreign Earned Income Exclusion（FEIE）** | $132,900（工资）| 海外工作日 > 330 天或 Bona Fide Residence |
| **Foreign Tax Credit（FTC）** | 无上限 | 外国已缴税抵扣美国税 |
| **Foreign Housing Exclusion** | 与 FEIE 联动 | 海外住房津贴部分豁免 |

**特殊权利**：
- 可继续供 [[401K]] / Roth IRA（需美国来源工资）
- 可做 Roth Conversion
- 全球资产面临美国遗产税（2026 豁免 $13.99M）

**放弃国籍后果**：
- 触发 [[Exit-Tax]]
- 后续 [[Section-2801遗产]] 40% 税
- 办理费 $2,350

### 2. Green Card Holder / LPR（绿卡 / 永久居民）

**税务义务**：**与公民相同**
- 全球所得申报
- 完整 FBAR / 8938 / PFIC
- **离开美国生活仍要报税**（Green Card Test 触发）

**独特陷阱：Long-Term Resident（LTR）**：
- 持绿卡 ≥ 8 个税年 → LTR
- 放弃时触发 [[Exit-Tax]]
- 参见 [[LTR-8年窗口]]

**丢失 / 放弃绿卡的影响**：

| 情形 | 移民身份 | 税务身份 |
|------|--------|---------|
| 物理丢失（卡片遗失） | 仍有效 | 仍是居民 |
| 长期离境（> 1 年无 Reentry Permit） | USCIS 可认定放弃 | **税务上不自动失效**，直到 I-407 |
| I-407 主动放弃 | 正式失去 | 失去（如 LTR 触发 Exit Tax） |
| [[Tie-Breaker-Rule]] 声明外国居民 | 保留绿卡 | **视为 Expatriation**（见 [[绿卡Tie-Breaker陷阱]]） |

**规划启示**：
- 绿卡 6-7 年是关键决策期
- 不长住 → 8 年前放弃（避 LTR）
- 长住 → 接受 LTR + 规划 Exit Tax

### 3. Resident Alien（非绿卡税务居民）

**如何成为**：通过 [[Substantial-Presence-Test]]
- 当年 ≥ 31 天 + 加权 3 年 ≥ 183 天

**税务义务**：**与公民 / 绿卡相同**
- 全球所得申报
- FBAR / 8938 / PFIC

**常见人群**：
- H1B、L1、O1 等非移民工作签证长期居美
- 中国派美国长期出差（"劳务派遣"）
- 合格 F1 学生超出 5 年豁免期

**优势（相比绿卡）**：
- **可用 [[Closer-Connection-豁免]]** 当年脱离（绿卡不可）
- **可用 [[Tie-Breaker-Rule]]** 安全声明外国居民（绿卡有陷阱）
- **非 LTR**：8 年前回国 / 更换签证 → 无 Exit Tax
- 可每年灵活选择居民身份

**实务策略**：
- H1B 转绿卡前 → 完整清理 PFIC
- 不转绿卡 + 离境 → 避免 LTR 风险
- 持续 RA 身份 + 回国计划 → 用 Tie-Breaker 灵活脱离

### 4. Nonresident Alien（非居民）

**如何成为**：未触发 Green Card Test 或 Substantial Presence Test

**税务义务（有限）**：仅美国来源收入
- 美国工资（1040NR）
- 美国房产租金（可选 NET election）
- 美国股票分红利息（预扣制）
- **美国房产卖出：FIRPTA 30% 预扣**

**美国预扣税率**：

| 收入类型 | 默认 NRA 税率 | 中美协定税率 |
|---------|-------------|------------|
| 分红 | 30% | **10%** |
| 利息（Portfolio Interest 豁免） | 0% | 0% |
| 银行存款利息 | 30%（机构代扣） | **10%** |
| 特许权使用费 | 30% | **7-10%** |
| 房产租金（总额） | 30% | 30%（或 NET election 按净额 + 常规税率） |
| 资本利得（非房产） | 0%（美国税） | 0% |
| 资本利得（美国房产） | 30% FIRPTA 预扣 + 联邦税 | 常规 |

**不能享受**：
- ❌ Roth IRA 供款（无 earned income 时）
- ❌ 401K 雇主 match（非雇员）
- ❌ Standard Deduction（大多数情况）
- ❌ 儿童税收抵免
- ❌ 美国 Marketplace 健保补贴

**优势**：
- ✅ 无 FBAR（不是美国人）
- ✅ 无 FATCA
- ✅ 无 PFIC
- ✅ **海外收入完全不管**

**常见人群**：
- 中国父母（持美国亲属签证或 B-2 短期）
- 中国投资美国房产
- 中国投资美国股票（非居民户头）
- EB-5 申请人等待期

## 身份对比速查表

| 维度 | 公民 | 绿卡 | 税务居民 | 非居民 |
|------|------|------|---------|--------|
| 全球所得申报 | ✅ | ✅ | ✅ | ❌ |
| FBAR | ✅ | ✅ | ✅ | ❌ |
| Form 8938 (FATCA) | ✅ | ✅ | ✅ | ❌ |
| [[PFIC]] | ✅ | ✅ | ✅ | ❌ |
| FEIE | ✅ | ✅ | ✅ | ❌ |
| FTC | ✅ | ✅ | ✅ | 有限 |
| [[Closer-Connection-豁免]] | ❌ | ❌ | ✅ | N/A |
| [[Tie-Breaker-Rule]] 安全 | ❌（Savings Clause）| ❌（7701(b)(6)(B)陷阱）| ✅ | N/A |
| Exit Tax 风险 | 放弃时 | LTR（8 年+） | 无 | 无 |
| Standard Deduction | ✅ | ✅ | ✅ | ❌ 大多 |
| [[Roth-IRA]] 供款 | ✅ | ✅ | ✅ | 需 earned income |
| 美国遗产税 | 全球资产 | 全球资产 | 全球资产 | **仅美国资产** |

**实务：自检身份**：
```
□ 持美国 Passport？ → US Citizen
□ 持有 I-551（绿卡）？ → Green Card Holder（无论住哪）
□ 过去 3 年加权 > 183 天且当年 ≥ 31 天？ → Resident Alien
□ 以上都否？ → Nonresident Alien
```

**关键策略对比**：

### 长期定居美国
- 身份：公民 或 绿卡（≥ 8 年 LTR）
- 税务：完全美国化
- 投资：可用 Roth / 401K / HSA

### 长期定居中国（打算回国）
| 身份 | 最佳路径 |
|------|---------|
| 公民 | 接受 Dual Resident + FTC（不能脱离）|
| 绿卡 < 8 年 | 7-8 年前 I-407 放弃 |
| 绿卡 ≥ 8 年 LTR | 接受 Exit Tax + Form 8854 |
| RA（非绿卡） | Tie-Breaker + Form 8833 安全脱离 |
| NRA | 保持 NRA |

### 中美双居（常年往返）
| 身份 | 最佳路径 |
|------|---------|
| 公民 | FTC，不能脱离 |
| 绿卡 | 接受 Dual Resident + FTC，**绝对不用 Tie-Breaker** |
| RA（非绿卡） | 当年 < 183 天 → Closer Connection；≥ 183 天 → Tie-Breaker |

### 配偶身份混合
- 常见：公民夫 + 非居民妻 / 绿卡夫 + 非居民妻
- 选项：
  - **MFJ + Section 6013(g) 选举**：非居民配偶选择按居民申报（双方全球所得）
  - **MFS**：各自申报（非居民不用 FBAR / 8938）
  - **Head of Household**（若有美国公民子女）
- 税负差异大，需模拟比较

## 特殊情况：Dual-Status Alien

年内身份发生变更的人（见 [[Dual-Status-Alien]]）：
- 居民期按居民规则报
- 非居民期按非居民规则报
- 双份表格（1040 + 1040NR）合并
- 首年 / 末年最常见
