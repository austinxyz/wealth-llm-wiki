## 定义

**Foreign Tax Credit（FTC，境外税收抵免，Form 1116）**：美国人在外国已缴所得税，可**$1 对 $1 抵扣**美国税。是解决双重征税的**最核心工具**——比 [[中美税收协定]] 的优惠更实用，比 [[Dual-Resident]] Tie-Breaker 更广适用。**保留绿卡回国场景下，FTC 几乎是唯一的减税机制**。

## 核心要点

**两种形式（几乎总选 Credit）**：

| 形式 | 效果 | 何时选 |
|------|------|-------|
| **Tax Credit**（首选）| **$1 外国税抵 $1 美国税** | 几乎所有情况 |
| Tax Deduction | 作 itemized deduction，仅省 22-37% × 外国税 | 极罕见（高税阶可略优）|

**四个必要条件**（缺一不可）：
1. 该税必须是**所得税性质**（Income Tax 或 in lieu of）
2. 必须是**合法且实际已缴**
3. 必须是**你本人的义务**
4. 必须是**外国政府**征收

**✅ 符合 FTC 的外国税**：
- 中国个人所得税
- 中国股息 / 利息预扣税（10%）
- 中国房租的个人所得税（10-15%）
- 中国资本利得税（20%）
- 加拿大、欧洲等国所得税
- 协定降税率的预扣税

**❌ 不符合**：
- 外国 **VAT / 增值税 / 消费税**
- **房产税**（外国）
- **社保 / 医保**缴费（除非有社保协定）
- 外国罚款 / 利息
- 退保费（保险）

### FTC 限制公式（核心）

```
FTC 实际抵扣 = min(
    外国已缴税,
    美国税 × (外国来源收入 / 全球应税收入)
)
```

**含义**：FTC **不能用来抵 "美国来源收入"的美国税**。

**算例**：
```
美国应税全球收入：$100,000
  - 美国来源：$70,000
  - 中国来源：$30,000
美国税总额：$18,000
中国已缴税：$6,000

FTC 上限 = $18,000 × ($30,000 / $100,000) = $5,400
实际抵扣 = min($6,000, $5,400) = $5,400

多缴的 $600 → 可**结转**（后递 10 年，前递 1 年）
```

**当外国税率 < 美国税率**（典型"低税国"）：
- FTC 可完全抵消
- 无结转余额

**当外国税率 > 美国税率**（典型"高税国"如中国顶格 45%）：
- FTC 有上限 → 超出不可抵
- 结转 10 年或流失

### 结转规则

**未用完的 FTC**：
- **前递 1 年**（carry back 1）
- **后递 10 年**（carry forward 10）
- 每年按 **category 独立**追踪
- 过期未用 → 永久流失

**高税国居民的典型困境**：
- 中国工资税高（顶格 45%）>> 美国联邦 22-24%
- FTC 每年超额结转
- 结转越积越多但可能用不完
- 10 年内若返美高收入可消化结转

### 收入分类（Category）⚠️ 关键复杂度

**FTC 按 category 独立计算**，不能跨类别抵免：

| Category | 典型收入 | 你的典型场景 |
|---------|---------|-----------|
| **Passive Income** | 利息、分红、资本利得、租金 | 中国银行利息、A 股分红、房租 |
| **General Category** | 工资、自雇收入 | 中国工作工资、自雇 |
| **GILTI** | 全球无形低税收入（企业）| 企业主 |
| **Branch Income** | 海外分支经营 | 跨境创业 |
| **Re-sourced by Treaty** | 协定重定来源 | 特殊情况 |

**关键**：
- **每个 category 独立 Form 1116**
- 中国工资 + 中国利息 → **两张 Form 1116**
- 不能"混抵"

**算例**：
```
中国工资 $80,000 → 中国税 $8,000（General category）
中国利息 $500 → 中国预扣 $50（Passive category）

不能合并。各自独立：
  Form 1116 General：计算工资相关 FTC
  Form 1116 Passive：计算利息相关 FTC
```

### 申报机制

**填表**：
- Form 1116（每 category 一份）
- 附在 1040 税表后

**必需信息**：
- 外国来源收入金额
- 已缴外国税（原币 + 换算 USD）
- 用什么汇率（通常收到日或年终）
- 分类到哪个 category
- 限制公式计算

**专业服务费**：
- 简单 1 张（仅 Passive）：$200-400
- 多张（Passive + General）：$500-1,000
- 复杂（GILTI / 信托）：$1,000-3,000

### 与 FEIE 的关系（参见 [[FTC-vs-FEIE选择]]）

**两大减税工具并存**：
- **FEIE**（Foreign Earned Income Exclusion）：**排除** ~$132,900 海外工资
- **FTC**：**抵免**外国已缴税
- 可以**组合使用**（FEIE 先排除 + 超出部分 FTC）

**大致规则**：
- 低税国海外工作 → FEIE 优
- 高税国（如中国）或被动收入 → FTC 优
- 在美国境内居住 + 中国被动收入 → **只能 FTC**（FEIE 要求海外居住）

### 简化规则（参见 [[FTC简化规则]]）

**$300 / $600 简化**：
- 若所有外国税 < $300（Single）/ $600（MFJ）
- 且全部来自 Passive + 1099-DIV / 1099-INT
- **免填 Form 1116**，直接在 1040 Line 抵免
- 实用场景：美国券商买 VXUS，1099-DIV 上有 "Foreign Tax Paid: $250"

### 中国场景实操

**场景 A：美国居住 + 中国被动收入**（最常见）
```
中国银行利息 $1,500（预扣 10% = $150）
中国 A 股分红 $3,000（协定 10% 预扣 = $300）

合计外国税 = $450 < $600 MFJ 门槛
→ **免 Form 1116**（直接抵免）
```

**场景 B：美国居住 + 中国房租**
```
中国房产租金 $12,000/年
中国个人所得税 $1,200

→ **需要 Form 1116**（Passive category）
→ 按限制公式计算实际抵扣
```

**场景 C：中国居住 + 中国工资（保留绿卡）**
```
中国工资 $80,000
中国个人所得税 $12,000

选项 1：FEIE 排除（Form 2555）
  $80k < $132,900 → 全部排除
  不用 FTC

选项 2：FTC（Form 1116 General）
  按限制公式计算

选项 3：组合（若工资 > $132.9k）
  FEIE 排除前 $132,900
  超出部分用 FTC
```

**场景 D：退休回国 + 401K 提取 + 中国全球征税（第 7 年后）**
```
美国 401K 提取 $100k → 美国税 $22k（22% 联邦）+ $9.3k（CA）= $31.3k
中国税（若触发 [[中国6年规则]] 第 7 年）：$20k（20% 被动）或按综合

FTC 应用：
  美国税 $31.3k
  中国税 $20k
  FTC = min($20k, 美国税 × 中国源比例) = ?
  
  关键：401K 提取算"中国源"还是"美国源"？
    - 若算美国源 → 中国征税，美国征税，FTC 抵扣困难
    - 若算中国源（按协定 Re-sourced）→ FTC 可用

→ 需专业 CPA 确定 sourcing（技术性高）
```

### 中美协定的"Re-source" 特殊条款

**协定 Article 22** 允许：
- 某些美国源收入（如 401K 提取）
- **为 FTC 目的重新视为中国源**
- 填 Form 1116 **Category "Certain Income Re-sourced by Treaty"**
- 允许中国税抵免美国税（即使原本是美国源）

**实务意义**：回国场景的 401K / IRA 提取可能通过协定 Re-source 获得 FTC 抵扣。

### 常见错误

- ❌ **"不报收入就不用抵"**：必须报，但可抵免；不报 = 违规 [[FBAR]] + [[FATCA]]
- ❌ **"FTC 全额抵免"**：看限制公式（美国税 × 外国收入比例）
- ❌ **"所有外国税合并算"**：必须按 category 独立
- ❌ **"PFIC 的税也抵"**：[[PFIC]] 收入的 FTC 有额外限制，需专业师
- ❌ **"FEIE 总优于 FTC"**：情况各异，中国工资 > $132k 时 FTC 更好
- ❌ **"FTC 不用填 1116"**：除非满足 $300/$600 简化，否则必填

### 对高净值家庭的战略意义

**保留绿卡 + 回中国退休** 场景下：

```
若触发 [[中国6年规则]] 第 7 年全球征税：
  中国对全球所得征税（含美国 401K、美股分红、美国房租）
  
FTC 作为双重征税的唯一减税工具：
  中国税 ≤ 美国税 → FTC 完全抵 → 净税 = 美国税（未多交）
  中国税 > 美国税 → FTC 部分抵 → 多交 = 差额（无补救）

关键：中国税率通常 ≥ 美国税率（45% vs 37%）
  → 多数情况仍有净多交（保守估 +$5-15k/年）
  → 结转但可能用不完
```

**低税阶退休者**（美国税率 12-15%，中国税率 20%）：
- FTC 抵扣后仍多交 5-8 个百分点
- 但比**不用 FTC 完全双重征税**省一半

### Form 1116 逐步填写

**Part I**：
- 外国来源收入（按 category）
- 可减费用（跨境税务律师费、换汇费用等）

**Part II**：
- 外国已缴税（原币 + USD）
- 按年度或实际缴纳日

**Part III**：
- 限制公式计算
- 结转余额追踪
- 最终可抵扣金额

**Part IV**（结转）：
- 前递 1 年
- 后递 10 年
- 按 category 分别追踪

### 结转管理

**"FTC Carryover" 追踪表**（多年使用）：
```
Year | Category | 已缴外国税 | FTC 上限 | 实际抵扣 | 超出结转 | 来源年
2026 | Passive  | $1,200    | $800    | $800    | $400    | 2026
2027 | Passive  | $1,100    | $900    | $900    | $200    | 2027
                                      加上 2026 结转 $400
                                      总可用 = $1,300
                                      上限 $900 → 仍超 $400
                                      结转 $400（2027 来源）
```

### 专业工具

**软件支持**：
- TurboTax（支持简单 FTC）
- H&R Block Expat（专做跨境）
- **Taxes for Expats**（专业跨境服务 $450-1,500）
- **Greenback Expat Tax**（$495+ 基础）

**建议**：金额 > $1,000/年或多 category → 专业服务
