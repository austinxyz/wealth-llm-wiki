## 定义

**Closer Connection Exception（紧密联系豁免）**：即使数学上满足 [[Substantial-Presence-Test]]，若当年 < 183 天且能证明与某外国"联系更紧密"，仍可通过 **Form 8840** 声明为非居民，不需申报美国全球所得。是陪读家长、短期商旅、中美双居人群**最常用的 SPT 脱离工具**。

## 核心要点

**生效四要件**（全部需满足）：

1. **当年 < 183 天**在美国（即使加权数学满足 SPT）
2. 在某外国（通常是中国）维持 **"tax home"**（税务家）
3. 与该外国有**比美国更紧密的联系**
4. **及时提交 Form 8840**（随当年税表或独立提交）

**"紧密联系"的 IRS 判断因素**：

| 维度 | 应在外国方证据 |
|------|--------------|
| 永久居所位置 | 外国有自有/长租房产 |
| 家庭住所 | 配偶、子女主要在外国 |
| 工作中心 | 主要雇主 / 自雇在外国 |
| 银行账户 | 主要账户、工资代发在外国 |
| 券商账户 | 投资主账户在外国 |
| 驾照、选民登记 | 在外国 |
| 俱乐部会员 | 在外国 |
| 宗教、政治、文化参与 | 在外国 |
| 医疗保险 | 主要健保在外国 |
| 个人物品存放地 | 家具、车辆、日常用品在外国 |
| 葬礼规划 | 墓地在外国 |

> 💡 IRS 倾向看"主要生活重心在哪"，不是单一因素。越多证据在外国，成功率越高。

**典型适用场景**：

### 场景 1：陪读家长（B-2，最常见）
- 中国退休夫妻每年来美 4-5 个月陪伴孙辈
- 持续数年触发 SPT
- **Form 8840 声明**：主要住所、家庭、资产、医保在中国 → 成功豁免
- 前提：名下中国房产仍在、仍有中国社保医保、主要生活在中国

### 场景 2：商务人士短期派驻
- 中国高管来美 4-6 个月项目
- 当年 < 183 天
- 家人仍在中国
- Form 8840 + 中国雇主在职证明 → 豁免

### 场景 3：投资人考察
- 多次短期来美看投资项目/EB-5
- 控制年度 < 183 天
- 配合紧密联系文件（物业、家庭）

**不适用的场景**：
- ❌ 当年 ≥ 183 天（数学硬伤，只能走协定 Tie-Breaker）
- ❌ 绿卡持有者（自动税务居民，必须 I-407 放弃）
- ❌ 正在申请美国绿卡（申请行为本身违反"紧密联系"原则）
- ❌ 在美国购置自住房并实际居住
- ❌ 配偶/子女主要在美国

**与税收协定 Tie-Breaker 的区别**：

| 维度 | Closer Connection | Tax Treaty Tie-Breaker |
|------|------------------|----------------------|
| 依据 | 美国国内税法 | 双边税收协定（如中美） |
| 当年天数上限 | **< 183 天** | 可以 ≥ 183 天 |
| 表格 | Form 8840 | Form 8833（Treaty-Based Return Position） |
| 判定规则 | IRS 单边看紧密联系因素 | 协定层级规则（永久住所 → 重要利益中心 → 惯常居所 → 国籍） |
| 难度 | 相对简单 | 专业 + 协定解读 |
| 适合 | 当年 < 183 天的短居者 | 长居者或绿卡放弃前 |

**Form 8840 提交要点**：
- **表格**：IRS Form 8840（Closer Connection Exception Statement for Aliens）
- **截止日**：通常 6 月 15 日（与 Form 1040NR 同步）
- **独立提交**：若不需报美国非居民税表，可单独寄
- **连续声明**：每年触发 SPT 都要重新提交
- **追溯**：错过当年 → **不能追溯**（IRS 要求及时性）

**Form 8840 关键问答**：
- 列出 3 年美国天数
- 列出紧密联系"tax home"所在国
- 回答 10+ 个生活重心问题（住所、家庭、银行、俱乐部等）
- 签名 + 地址

**保留文件（IRS 审查时用）**：
- 中国房产契约、物业费发票
- 中国社保医保缴费记录
- 中国银行月结单
- 中国驾照副本
- 中国雇主劳动合同 / 工资单
- 中国纳税记录（个税 APP 截图）
- 飞机票、护照盖章

**常见失败原因**：
- **当年意外超 183 天**（如因健康延误）—— 此时需健康原因"排除天数"+ 走 Tie-Breaker
- **材料不足**—— IRS 审查时缺乏中国生活重心证据
- **多年在美购置资产**（如美国自住房）—— 联系反向
- **年年都卡 182 天**—— IRS 可能视为"规避居民身份"加强审查

**Form 8840 vs 不申报的税务成本**：
- 不提交 → 被动按 SPT 规则变税务居民 → 全球所得应税
- 典型中国高净值家庭：多 $100k-1M 税 + [[FBAR]] + [[PFIC]] 罚款
- Form 8840 **几乎零成本**（自己填或 CPA $200-500）—— ROI 极高

**与 Exit 规划的关系**：
- 未来若打算正式移民 → **早期**的 Closer Connection 记录不影响
- 未来若打算 LTR 后放弃 → 与 [[Exit-Tax]] 的 LTR 定义无关（SPT 年数不计入 LTR 8 年）

## 与其他概念的关系

- [[Substantial-Presence-Test]]：本豁免是 SPT 触发后的脱离机制
- [[天数计算规则]]：确定当年是否 < 183 天的前提
- [[双重居民]]：协定 Tie-Breaker 是 Closer Connection 之外的备选路径
- [[FBAR]]：成功豁免则无需申报
- [[PFIC]]：成功豁免则避开中国基金地雷
- [[Exit-Tax]]：Closer Connection 年不计入 LTR 8 年窗口
- [[双边资产策略]]：豁免身份下可保持中国资产不触发美国税务
