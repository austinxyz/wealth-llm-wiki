## 定义

**退休后从美国回中国居住**是许多华裔退休者的梦想，但税务复杂度**极高**——美国税务义务是否消失？中国是否要征税？Social Security / Medicare / 401K / Roth IRA 如何处理？资产移动和继承？**每个决策都依赖身份、停留时长、资产结构**。本条目是退休回国的综合决策框架，链接所有相关深度主题。

## 核心要点

**五大核心议题**：

```
1. 美国税务义务是否消失？（公民/LTR 几乎不消失）
2. 中国是否要征税？（第 7 年起可能全球征税）
3. Social Security / Medicare 怎么办？
4. 401K / IRA / Roth 如何领取？
5. 资产移动和继承怎么办？
```

**"回去多久"决定一切**：

| 回国时长 | 策略重点 |
|---------|---------|
| **< 半年/年** | 与在美国居住类似；保留所有美国账户 |
| **约 6 个月/年** | SPT / 非居民临界；保留所有美国选项 |
| **> 6 个月/年，< 6 年** | 中国税务居民，但仅境内所得征税（[[中国6年规则]]）|
| **> 6 年** | 中国对全球所得征税，重大税务调整 |
| **永久回国** | 考虑放弃身份、清理资产 |

### 1. 美国税务义务的去向

**身份决定一切**：

| 身份 | 美国税务义务 |
|------|------------|
| 美国公民 | 全球所得永久申报（仅放弃国籍可脱离）|
| 绿卡（< 8 年 LPR）放弃 | 放弃后脱离，仅美国源收入按 NRA 规则 |
| 绿卡（LTR ≥ 8 年）放弃 | 触发 [[Exit-Tax]]，之后脱离（除美国源外）|
| 绿卡不放弃 | 与公民相同：全球申报 |
| 非绿卡 RA 回国 | 失 SPT 即脱离，或用 [[Closer-Connection-豁免]] / [[Tie-Breaker-Rule]] |

### 2. 中国税务义务

**关键点：2019 新个税法 + [[中国6年规则]]**：
- 一个日历年在中国 ≥ 183 天 → 中国税务居民
- 头 6 年内，若每年有 **连续 30 天离境** → 仅境内所得征税
- **第 7 年起**未离境满足条件 → **全球所得**按中国税法征税
- 中国最高边际税率 **45%**（> 美国 37%）

**典型陷阱**：
- 绿卡 + 住中国 7+ 年 + 每年不离境 30 天
- 中国对 401K 提取、美股分红、美国房租**全征税**
- 美国同样全球征税
- 只能靠 [[Foreign-Tax-Credit]] 抵消（部分，不完全）

### 3. Social Security 跨境领取

**基本规则**：
- 美国公民 → 几乎任何国家可领取
- 绿卡 → 可领取但有限制
- **中国特殊**：SSA 不在中国设机构，需通过美国银行账户中转

**常见方案**：

| 方案 | 说明 |
|------|------|
| 保留美国银行（Direct Deposit）+ Wise 转回中国 | 最常用 |
| 国际银行（HSBC HK、Schwab International）| 跨境方便 |
| 邮寄支票 | 慢 + 易丢失，不推荐 |

**税务处理**：
- SS 按 Combined Income 计算美国应税比例（0/50/85%）
- 中国可能也征（中美协定 Article 17 有争议）
- **加州离开后加州不再征**（加州本来就不征 SS）

**持绿卡长期离境 > 6 个月的风险**：可能影响继续领取，需 Reentry Permit。

### 4. Medicare 海外无用

**残酷现实**：Medicare **不覆盖**美国境外医疗（极少例外）。

参见 [[Medicare海外策略]] 详细方案对比。

**简要对比**：

| 选项 | 月费 | 回美保底 | 适合 |
|------|------|---------|------|
| 继续缴 Part B | $202.90+（2026）| ✅ | 可能回美者 |
| 停缴 Part B | $0 | ❌ 回美 10%/年罚款 | 完全不回美 |
| 中国商业大病险 | ¥30-80k/年 | 中国端 | 不回美 + 中国长居 |
| 国际健康险（Cigna Global 等） | $2-8k+/年 | 两边可用 | 常年两地奔波 |

### 5. 退休账户提取

**美国端**：

| 账户 | 美国税 |
|------|--------|
| Traditional 401K / IRA | 普通所得税率（退休期可能较低）|
| Roth 401K / Roth IRA | 满足条件完全免税 |
| Annuity | 按协定 Article 17 规则 |

**中国端**（第 7 年后若触发全球征税）：

| 账户 | 中国税可能情形 |
|------|-------------|
| Traditional 401K / IRA 提取 | 中国可能对提取征税（Article 17 pension 条款争议） |
| **Roth IRA 提取** | **中国不承认免税性质，可能全额按 20% 被动所得征税** ⚠️ |
| 美国股票分红 | 20% 中国税 + 美国协定 10% 预扣 → FTC 抵 |
| 美国房产租金 | 10-15% 中国税 + 美国税 |

**Roth 陷阱是最大的反直觉**：
- 美国端免税的 Roth IRA
- 中国端仍按普通所得税（或被动 20%）
- 原本的"免税优势"在中国**可能失效**
- 详见 [[Roth在中国的税务陷阱]]

### 6. 资产配置调整

**回国前应处理**（见 [[回国前资产清理]]）：
- PFIC 清理（5 年分摊亏损）
- 低收入年实现长期资本利得（0%/15%）
- Roth Conversion（利用 Ladder）
- 房产决策（卖 vs 持；Section 121 $500k 豁免）
- 美国银行 / 券商账户保留

**回国后**：
- ❌ 不要买中国公募基金 / ETF / 理财 / 银行保险（全是 [[PFIC]]）
- ✅ 中国**个股**可以（非 PFIC）
- ✅ 香港 ADR 可以
- ✅ 美国账户保留使用

### 7. 继承规划

**美国遗产税豁免**：

| 身份 | 2026 豁免 | 范围 |
|------|---------|------|
| 美国公民 / LPR | $13.99M | **全球资产** |
| 非居民外国人（NRA）| **$60,000** | 仅美国境内资产 |

**配偶条款**：
- 美国公民配偶 → **无限**婚姻豁免
- 非公民配偶 → 有限（需 Qualified Domestic Trust QDOT）

**跨境继承**：

| 场景 | 税务 |
|------|------|
| 美国资产 → 中国继承人 | 正常美国遗产税流程 |
| 中国资产 → 中国继承人 | 中国无遗产税（2026 尚无立法） |
| Covered Expatriate → 美国受益人 | **40% 税**（[[Section-2801遗产]]） |

## 决策框架：五大核心问题

```
问题 1：是否放弃绿卡 / 公民？
  - 绿卡 < 8 年：可放弃避 Exit Tax
  - 绿卡 ≥ 8 年 LTR：放弃代价大，接受 FTC
  - 公民：几乎不放弃（除非极高净值 + 彻底不回美）

问题 2：是否保留 Medicare？
  - 可能回美治疗 → 保留 Part B
  - 完全不回美 → 停缴 + 中国商业险

问题 3：PFIC 如何处理？
  - 回国前清理所有中国基金
  - 回国后不再买

问题 4：资产配置是否调整？
  - 低收入年实现长期资本利得
  - Roth Conversion 在美国税率低时完成
  - 房产决策结合 Section 121 豁免

问题 5：继承规划是否调整？
  - 非 US 配偶 → 考虑 QDOT
  - Covered Expatriate → Section 2801 影响
  - 遗产分配考虑中美两地
```

## 时间线建议

**回国前 5-10 年**：
- 规划绿卡 8 年窗口（若不想成为 LTR）
- 最大化 401K / HSA / Roth 供款
- PFIC 清理分 5 年
- 低收入年 Roth Conversion

**回国前 2 年**：
- 确定放弃身份 vs 保留
- 清理所有中国基金
- 资本利得实现窗口
- 咨询跨境 CPA + 移民律师 + 遗产律师

**回国当年**：
- 处理 Medicare 决策
- 设置美国银行直接存款
- 建立 SS 接收渠道
- 若放弃：Dual-Status Return + Form 8854

**回国后每年**：
- 追踪中国天数（6 年规则）
- 每年 30 天连续离境（重置 6 年）
- 美国 1040 + FBAR 合规
- 中国个税申报

## 专业咨询建议

"退休回国"是**极复杂**跨境规划，需要三方同时咨询：

| 专业 | 内容 |
|------|------|
| 跨境 CPA（美中双执照） | 税务申报 + FTC + Treaty |
| 移民律师 | 绿卡放弃 / Reentry Permit / 身份规划 |
| 遗产律师 | QDOT / Section 2801 / 跨境继承 |
| 保险顾问 | Medicare / 中国商业险 / 国际健康险 |

**费用参考**：首次综合规划 $5,000-20,000；每年合规 $3,000-10,000。
