## 定义

**Form 8621（Information Return by a Shareholder of a Passive Foreign Investment Company or Qualified Electing Fund）**：美国人持有 [[PFIC]] 时**每年每只基金**必须单独提交的 IRS 表格。内容：PFIC 身份、本年选择（Default/MTM/QEF）、分红、卖出、Excess Distribution 计算等。未报 → 税务追诉期**无限延长** + **$10,000+ 罚款**。是 PFIC 合规的核心载体。

## 核心要点

### 触发：何时必须提交 Form 8621

**每年**必须提交 Form 8621，若你是美国人且：
1. **持有 PFIC 股份**，**任何金额**（无门槛）*
2. 或在 PFIC 中**实现直接或间接利益**
3. 或从 PFIC 收到**分红**
4. 或**卖出** PFIC 股份（当年必报）
5. 或做 **QEF / MTM election**

*技术上有一个 **"de minimis" 豁免**（< $25,000 总值），但实务中不推荐依赖。

**每只 PFIC 一份表格**：
- 3 只中国基金 → **3 份 Form 8621**
- 若家庭共有多只 → 按法定所有权分配

### 表格结构

**Form 8621 的主要部分**：

| 部分 | 内容 |
|------|------|
| **Part I** | 基本信息（PFIC 名称、税号、你持股 %）|
| **Part II** | **QEF Election** 或相关选择 |
| **Part III** | **Mark-to-Market** Election |
| **Part IV** | **Excess Distribution** 计算（Default 情况）|
| **Part V** | QEF 的**年度所得**披露 |
| **Part VI** | Income / Gain 汇总 |

### 申报机制

**附在 1040 税表后**：
- 不是独立提交
- 电子申报（大多数税务软件支持）
- 纸质邮寄（若用 TurboTax 等消费级软件，可能需手动附加）

**截止日期**：与 1040 同（4/15 或延期 10/15）

**必需信息**：
- PFIC 全名 + 地址
- 基金公司税号（若有）
- 你的持股 %
- **持仓价值**（年末）
- **Cost Basis**
- 年度分红金额
- 卖出详情（若有）

### 选择（Election）的机制

**三种选择对应不同 Part**：

#### 1. Default（什么都不选）
- 填 **Part IV**（Excess Distribution）
- 公式复杂：
  ```
  当年实际分红
  - 过去 3 年平均分红 × 125%
  = 超过部分 = "Excess Distribution"
  
  + 卖出时资本利得 → 全部视为 Excess Distribution
  
  → 分摊到持有全部年数
  → 每年部分按当年最高普通税率（37%）
  → 加 IRS 利率（~8%）复利利息罚款
  ```

#### 2. Mark-to-Market（MTM）
- 填 **Part III**
- **首次选择**必须是**取得 PFIC 的第 1 年**
  - 过期则需**"Purging Election"**（先按 Default 清算过去，才能改 MTM）
- 每年按市价标记
- 增值按普通所得
- 减值按普通损失（受限）

#### 3. QEF（Qualified Electing Fund）
- 填 **Part II + V**
- 需基金提供 **Annual Information Statement (AIS)**
- 中国基金**极少**提供
- 股东按 pro-rata 报告：
  - Ordinary earnings
  - Net capital gain（保留长期资本利得税率）

### Form 8621 的复杂度

**为何"简单"表格实际极难**：

**1. Excess Distribution 计算**：
- 需过去 3 年每年分红记录
- 分摊算法依持有天数
- 利息罚款按 IRS 汇率复利（每年变化）
- 手算易错，**必须用专业软件**

**2. MTM 第一年估值**：
- 需要取得日 FMV
- 若多年前买 → FMV 难查
- 中国基金的"净值"可能非 IRS 认可的 FMV

**3. Pro-Rata 分配**（QEF）：
- 需基金月度 / 季度数据
- 中国基金不按美国 GAAP 披露
- 即使拿到 AIS，翻译 + 重算工作量大

**4. Form 8621 与其他表格的联动**：
- **1040 Schedule D**：QEF 的净资本利得
- **1040 Schedule B**：QEF 的 ordinary earnings
- **Form 8938**：PFIC 资产列出
- **FBAR**：PFIC 所在账户

### 罚款与追诉期

**未报 Form 8621**：

| 后果 | 详情 |
|------|------|
| **$10,000** 初始罚款 | IRS 核查时发现 |
| 通知 90 天后仍未报 | 每 30 天 +$10,000，最高 $50,000 |
| **税务追诉期无限延长** | 整个 1040 保持敞开状态 |
| 联动 [[Form-8938]] / [[FBAR]] 罚款 | 若相关资产也漏 |

**"税务追诉期无限延长"的严重性**：
- 正常 1040 追诉期 3 年
- 漏 Form 8621 → **无时效**
- IRS 可以在 10 年、20 年后仍追查
- 直到你主动申报 → 追诉期才开始计算

### 专业服务费用

**Form 8621 的 CPA 费用**：

| 情况 | 费用 |
|------|------|
| 简单 Default（小额分红） | $200-500 per form |
| 复杂 Default + Excess Distribution | $500-800 per form |
| MTM Election | $300-600 per form |
| QEF（有 AIS）| $500-1,000 per form |
| QEF（无 AIS，需自己算）| **$1,500-3,000** per form |

**典型场景费用**：
- 3 只中国基金（简单）：$600-2,400/年
- 5 只 + 多年补报：$5,000-15,000 一次性

### 补报 Form 8621

**发现历年漏报**：
- 用 [[FBAR补报程序]] Streamlined（SDOP / SFOP）
- Streamlined 覆盖 FBAR + 8938 + **Form 8621**
- 补 3 年税表（含 Form 8621）
- 5% SDOP 罚款（美国境内居民）

**若未发现但持续持有**：
- 仍每年继续补
- 越早处理越好
- **不要 Quiet Disclosure**（视为故意）

### 中国基金的实际操作建议

**短期持有者**（< 1 年）：
- 尽快卖出
- 当年填 Form 8621 Part IV（Default）
- 一次性申报，不再持有

**长期持有者**（> 2 年，想继续持有）：
- 做 **MTM Election**（当年首次最优）
- 每年 Part III 申报
- 减少 Default 下 Excess Distribution 爆雷风险

**有巨额 PFIC 的**（> $500k）：
- 强烈建议清理（一次性 Default 剧烈，但之后清净）
- 低收入年卖出
- 如无法清理 → MTM + 长期心理准备每年 $2-5k CPA 费

### 常见 Form 8621 陷阱

**陷阱 1：首年不选 MTM**
- 第 1 年默认 Default
- 想改 MTM 需做 Purging Election（先按 Default 清算过去年份）
- **建议**：首次发现 PFIC 就立即咨询 CPA 做 MTM

**陷阱 2：多年后才发现持有 PFIC**
- 不仅补 Form 8621
- 还要补 1040 修订（Excess Distribution 可能重新计算税）
- 通过 Streamlined 补齐最稳妥

**陷阱 3：支付宝余额宝用户完全不知**
- 以为是"活期"
- 实际持有天弘基金份额
- 多年未报 → Streamlined 补

**陷阱 4：夫妻间转移**
- "把基金转到配偶名下避税"
- 无效 → IRS 视为单一 US Person 家庭
- 夫妻都是美国人 → 都要报

**陷阱 5：公司 401K 中的外国基金**
- 极少数公司 401K 提供非美国基金
- 若选了 → 是 PFIC
- 但有例外：**合格退休账户内的 PFIC 通常豁免 Form 8621**
- 需确认计划文件

### 中国 QDII 基金的特殊性

**QDII 基金**（中国发行，投海外）：
- **发行主体是中国基金公司** → **仍是 PFIC**
- 即使底层投美股（Apple、Tesla）
- 别被名字迷惑："易方达全球蓝筹" 依然 PFIC
- 应改买美国直接基金（VTI、VOO）

### 避免 Form 8621 最佳策略

**三条黄金原则**：
1. **不买任何中国基金 / 理财 / 储蓄险**
2. **仅通过美国券商买美国基金**
3. **想要中国敞口？用美国 ETF（MCHI/KWEB/FXI/BABA）**

**零 PFIC 的生活**：
- 无需 Form 8621
- 年度合规成本 - $2-10k/年
- 心理负担 - 爆雷风险
