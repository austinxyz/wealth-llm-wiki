## 定义

**回国前资产清理**：计划长期回中国居住前，利用"仍是美国居民 + 美国退休后低收入"的窗口期，完成一系列**不可逆**的资产优化动作——PFIC 清理、长期资本利得实现、Roth Conversion、房产决策等。**时机错过就失去大量节税空间**，且回国后再做多数被封堵。

## 核心要点

**回国前的战略窗口**：
- **退休到放弃身份前**：通常 2-5 年
- 收入低（退休工资/SS）→ 长期资本利得可能 0% / 15% 档
- 尚未放弃身份 → 仍可使用美国税优账户
- 还未触发中国全球征税 → 美国端处理更灵活

**核心清理清单**：

### 1. PFIC 清理（最优先）

**问题**：
- [[PFIC]] = 中国公募基金 / ETF / 理财 / 银行保险的**税务地雷**
- 有效税率可达 **50-80%**（普通所得税率 + 利息惩罚）
- 一旦为美国税务居民 → 每年 Form 8621 每只基金
- 回国前**必须清理**

**操作**：
```
1. 盘点所有中国持有：
   - 公募基金（股票/债券/混合）
   - 银行理财（非保本浮动收益）
   - 储蓄型保险（有现金价值）
   - ETF
   - 货币基金（余额宝、零钱通等）

2. 全部卖出
   - 利用"5 年内分摊亏损"机制（PFIC Section 1291）
   - 美国端按 PFIC 规则申报 Form 8621
   - 中国端卖出可能无税（A 股长期持有免）

3. 回国后不再买
   - ✅ 可买：中国 A 股个股（非 PFIC）
   - ✅ 可买：香港 ADR
   - ✅ 保留：美国券商账户（继续买 VTI / VOO / BND）
   - ❌ 不买：中国基金 / ETF / 理财 / 储蓄险
```

### 2. 低收入年实现长期资本利得

**窗口**：
- 退休当年及之后，收入降低
- 2026 MFJ 长期资本利得 0% 上限：**MAGI < $96,700**
- MAGI < $96,700 部分的 LTCG → **联邦 0% 税**

**操作**：
```
# 年度 LTCG 收获（Tax Gain Harvesting）

例：2027 年（退休第 2 年）
  - 年度工资 $0
  - SS $30,000（假设 85% 应税 = $25,500 计入 AGI）
  - 预估其他收入 $10,000
  - AGI = ~$35,500
  - Standard Deduction = $31,500（MFJ 2026）
  - 可用 LTCG 0% 空间 = $96,700 - $35,500 = $61,200

→ 本年可**实现 $61,200 长期资本利得 + 0% 税**
→ 卖 VTI 实现利得后立即买回（无 Wash Sale 限制对盈利）
→ Cost Basis 重置到当前价格
→ 未来卖出时应税利得少 $61,200
```

**连续多年做**：3 年可重置 $180k+ Cost Basis，大幅降低未来税负。

### 3. Roth Conversion

**窗口**：
- 退休后工资 $0
- 在 22% 或 24% 联邦税阶上限内转换
- 利用低税阶"锁定"Roth 免税空间

**操作**：
```
# 退休低税率年 Roth Conversion

假设 2027 年：
  AGI 起点 = $35,500（SS + 小额其他）
  
目标：填满 22% 税阶（2026 MFJ 上限约 $220k）
  可 Roth Convert = $220,000 - $35,500 = $184,500
  
→ Traditional 401K / IRA 转 $184,500 到 Roth
→ 本年 22% 联邦税约 $40,600
→ 未来免税空间 $184,500 + 复利增长 → 20 年后 ~$710k（7%）
```

**梯形转换**（[[Roth-Conversion-Ladder]]）：
- 每年转换 → 5 年后可免罚取出
- 连续 3-5 年转换 → 大幅增加免税资产

**回国后的限制**：
- Roth Conversion 仍可做（技术上）
- 但**中国可能对转换年的"分配"征税**（Article 17 pension 争议）
- 不如回国前完成

### 4. 房产决策

**美国自住房**：

| 决策 | 考虑 |
|------|------|
| **卖出** | Section 121 夫妻豁免 $500k 资本利得（需"2 年中 5 年自住"）|
| **保留出租** | 租金收入美国应税 + 未来卖出失 Section 121 资格 |
| **保留作为 Pied-à-Terre** | 探亲 / 回美短居用 + 维护成本 |

**最佳窗口**：
- **卖出**：已满足 2/5 年自住 + 资本利得 < $500k 豁免 → 无税卖出 → 换 $1M 现金
- **资本利得 > $500k**：超出部分 LTCG 15-20% 税
- **保留出租**：连续 3 年出租后失去 Section 121（之后卖出超额增值全额应税）

**中国房产**：

| 决策 | 考虑 |
|------|------|
| **卖出** | 美国端资本利得应税（全球征税），无 Section 121 豁免 |
| **保留自住** | 回国后自住降低生活成本 |
| **保留出租** | 租金美国应税 + 中国所得税 10-15% |

**时间点**：中国房产卖出最佳在**已失去美国税务身份后**（非 LTR 绿卡 + 放弃后）—— 避免美国全球征税。

### 5. 集中股票风险减仓

**场景**（类似用户 EBAY $293k）：
- 公司老员工持有大额个股
- 未实现增值大（成本基 $100k，市值 $293k，利得 $193k）
- 集中度风险 + 未来税务风险

**回国前处理**：
- 利用低收入年 0% LTCG 档分批卖出
- 或一次性卖出实现 20% LTCG（州税最高 9.3% CA）
- 换成 VTI / VOO 分散

**回国后卖**：
- 若未放弃身份 → 美国全球征税 + 可能中国征税（第 7 年起）
- 若已放弃身份（公民不行）→ 脱离美国税，但仍可能中国税

### 6. 美国账户保留

**必须保留**：
- 美国券商（Fidelity、Vanguard、Schwab）
- 美国银行（BofA、Chase 等）
- Social Security 接收账户
- 医保（Medicare Part B 若保留）

**可关闭**（低优先）：
- 信用卡（保留 1-2 张有 0 外汇费的：Schwab、Chase Sapphire）
- 多余小额银行账户
- 本地公用事业账户

**回国后美国账户管理**：
- Mailing address → 家人 / 朋友 / 代理地址（某些州税优但不可 CA）
- 在线账户登录 → 可能需美国手机号（保留 Google Voice）
- 证件：保留驾照 + 护照

### 7. 继承文件更新

**回国前**：
- 更新 Will / Revocable Living Trust（[[Revocable-Living-Trust]]）
- Beneficiary Designations（[[Beneficiary-Designation]]）
- Medical Directive / Power of Attorney
- 考虑 QDOT（非公民配偶）

**回国后更新**：
- 每 3-5 年 review
- 重大家庭变化（孙辈出生 / 子女搬家）时更新
- 关注 [[Section-2801遗产]] 对 Covered Expatriate 受益人影响

### 8. HSA 的特殊处理

**[[HSA]] 在回国后的地位**：
- 账户可保留（美国机构持有）
- **合格医疗开销**：Medicare 保费可付（若保留）；中国医疗费用**无法合格**（非美国服务）
- 65 岁后非合格开销 → 按普通所得税（无 20% 罚款）
- **相当于降级为 Traditional IRA**

**回国前清理建议**：
- 尽量用满 HSA 报销美国期间的医疗收据（历年积累）
- 保留美国医疗开销收据（未来可随时报销，无时效）
- 若完全不回美 → 65 岁后按普通所得提取即可

```

  ✓ 开始 LTCG 收获（每年利用 0% / 15% 空间）
  ✓ 规划绿卡 8 年窗口（若未 LTR）

  ✓ Roth Conversion Ladder 启动
  ✓ 集中股票分批减仓
  ✓ 咨询跨境 CPA + 移民律师 + 遗产律师

  ✓ 房产决策（卖 vs 持）

  ✓ 最终资产盘点 + Form 8854 准备（若放弃）
```

## 常见错误

- ❌ **"等我回国再卖中国基金"** → 成为 PFIC 后税负暴增
- ❌ **"Roth 免税永久，不用管"** → 中国可能征税（[[Roth在中国的税务陷阱]]）
- ❌ **"美国自住房出租再卖"** → 失去 Section 121 $500k 豁免
- ❌ **"等下一年实现 LTCG"** → 错过 0% 档窗口
- ❌ **"放弃身份后再处理资产"** → 已失去美国税优账户优势
- ❌ **"HSA 等回国后继续供"** → 需 HDHP 保险 + 美国账户资格
