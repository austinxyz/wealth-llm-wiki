## 定义

**Tie-Breaker Rule（决胜规则）**：当一个人按两国各自国内法都是税务居民（[[Dual-Resident]]），中美税收协定 Article 4 提供的**5 层顺序规则**决定**唯一**协定居民国。用于打破双重居民僵局，但绿卡用须注意 [[绿卡Tie-Breaker陷阱]]。

## 核心要点

**5 层顺序（必须从第 1 层开始，满足即终止）**：

```
第 1 层：Permanent Home（永久住所）
    ↓ 两国都有 / 都无
第 2 层：Center of Vital Interests（重要利益中心）← 最常用
    ↓ 无法判定
第 3 层：Habitual Abode（惯常居所）
    ↓ 无法判定
第 4 层：Nationality（国籍）
    ↓ 国籍不定/双重国籍
第 5 层：Competent Authority Agreement（两国协商）
```

**第 1 层：Permanent Home（永久住所）**：

**判断标准**：
- 永久性质的住房（自有或长租）
- "Permanent" 强调**长期可用**，不只是短期居住
- 酒店、月租 Airbnb 不算
- 空置但长期保留的房产也可算

**典型案例**：
| 情况 | 永久住所在 |
|------|----------|
| 美国自住房 + 中国出租房 | 美国 |
| 中国老家祖宅 + 美国租房 | 中国 |
| 美中各有自住房 | **两国都有 → 下一层** |
| 两国都租房 | **两国都无 → 下一层** |

**第 2 层：Center of Vital Interests（重要利益中心）** ⭐ 最核心：

**综合判断因素**：

| 类别 | 具体内容 |
|------|---------|
| **个人关系** | 配偶、子女、父母所在 |
| **社会关系** | 朋友、社区、宗教组织 |
| **职业** | 主要雇主、生意所在 |
| **活动** | 政治、文化、宗教、俱乐部参与 |
| **收入来源** | 主要工资、经营、投资所得 |
| **财产所在** | 主要房产、金融资产 |
| **银行活动** | 主账户、信用卡 |

**权重并不均等**，IRS 倾向看"**生活重心**"（typical: 配偶+孩子+主要工作 > 其他）。

**中国 vs 美国的常见权衡**：

| 在美国 | 在中国 |
|-------|-------|
| 绿卡本身 | 户籍 / 社保 |
| 子女在美上学 | 父母在中国 |
| 美国工作 | 中国生意 |
| 美国房产 | 中国房产 |

**建议**：列出所有 Vital Interests 做成矩阵，哪边多 → 该边为中心。

**第 3 层：Habitual Abode（惯常居所）**：

**判断标准**：
- 实际居住时间更多的国家
- **看整体生活模式**（不只是 183 天）
- 长期习惯性居住（习惯的生活地）

**与 SPT 的区别**：
- SPT 是美国国内法的 183 天数学公式
- Habitual Abode 是协定层面的"惯常居住地"概念
- 可能 SPT 触发但 Habitual Abode 不在美国（如商旅密集但家在中国）

**第 4 层：Nationality（国籍）**：
- 到这层基本是"形式居民"（两国都只是形式居民）
- 看**公民身份**
- 美国公民 + 中国公民（双重国籍）→ 下一层

**第 5 层：Competent Authority Agreement**：
- 两国税务机关协商
- **极其罕见**
- 耗时 1-3 年，需专业代理

**中美协定 Article 4 原文精要**（中译意译）：
```
1. 在双方国内法下同为居民者
2. 应视为其"永久性住所"所在国居民
3. 如两国都有或都无 → 视为"重要利益中心"所在国
4. 如无法确定 → 视为"惯常居所"所在国
5. 如两国都是或都不是惯常居所 → 视为国籍国
6. 如双重国籍或无国籍 → 双方主管当局协商
```

**实务举例**：

### 例 1：绿卡 + 家在中国
- 绿卡 → 美国居民
- 中国住所 + 家人都在中国 → 中国居民
- 第 1 层：两国都有永久住所（美国自住房 + 中国家）
- 第 2 层：Vital Interests 在中国（配偶、父母、孩子都在）
- ✅ Tie-Breaker 结果：中国居民
- ⚠️ **但因绿卡 → 触发 [[Exit-Tax]]**（见 [[绿卡Tie-Breaker陷阱]]）

### 例 2：L1 在美工作但家在中国
- SPT → 美国居民
- 户籍 + 住所在中国 → 中国居民
- 第 1 层：美国租房（非永久），中国自住房
- ✅ Tie-Breaker 结果：中国居民（Article 4 第 2 项）
- ✅ 非绿卡 → 可安全用，通过 Form 8833 声明

### 例 3：美国公民回国退休
- 公民永远是美国居民（IRC）
- 住中国 → 中国居民
- 协定 Tie-Breaker **无效**（Savings Clause）
- 只能用 [[Foreign-Tax-Credit]]

**Form 8833 与 Tie-Breaker 的关系**：
- Tie-Breaker 决定你是协定上的哪国居民
- Form 8833 是**申报载体**（声明你依协定的居民国）
- 没 Form 8833 → Tie-Breaker 无效（每次 $1,000 罚款）

**IRS 审查重点**：
- Vital Interests 的证据链（配偶 / 孩子 / 工作 / 银行）
- 时间分配（护照盖章 + 航空记录）
- 连续 3-5 年的生活模式（不只是 1 年）
- 主观意图（言行是否指向"以 X 国为家"）

**关键文件准备**：
- 两国住所证明（契约、租约、物业费）
- 家庭成员住所证明
- 工作 / 收入证明
- 两国银行账户月结单
- 两国报税记录
- 社保 / 医保缴费记录
- 孩子学校 / 医疗记录
- 驾照、居住证
- 俱乐部 / 宗教会籍

**与 SPT 系统的互动**：
- Tie-Breaker 可以突破 SPT 的数学结果
- 例：SPT 加权 200 天 > 183 → 美国税务居民
- 但 Vital Interests 在中国 → Tie-Breaker 宣告中国居民
- **前提**：非绿卡 + 非公民

**Tie-Breaker 后的税务效果（若成功）**：
- 从 1040（全球所得）→ 1040NR（仅美国来源）
- FBAR / Form 8938 义务消失（若非美国人）
- 海外收入不在美国申报
- 仅美国来源工资、租金、股息按 NRA 规则课税
- **极大减轻合规负担**
