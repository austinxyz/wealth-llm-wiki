## 定义

**FTC（Foreign Tax Credit）vs FEIE（Foreign Earned Income Exclusion）**：美国人在海外工作/居住时的两大主要减税工具。**FTC** 抵扣外国已缴税；**FEIE** 排除 ~$132,900（2026）海外工资收入。两者**可组合**但有微妙互斥规则。**错选可能多交数千到数万美元税**。

## 核心要点

**两工具并存（可组合）**：

| 工具 | 机制 | 2026 额度 | 表格 |
|------|------|---------|------|
| **FTC** | 外国税抵扣美国税 | **无上限**（按限制公式）| Form 1116 |
| **FEIE** | 直接排除部分收入不计入 AGI | **$132,900**（工资性）| Form 2555 |
| **Foreign Housing Exclusion** | 排除部分住房津贴 | ~$18,700-42,000（视地点）| Form 2555 附 |

### 适用范围对比

| 收入 / 情况 | FTC | FEIE |
|------------|-----|------|
| 境外工资（Earned）| ✅ | ✅ |
| 自雇收入（海外）| ✅ | ✅（但需额外 SE 税 15.3%）|
| 境外利息、分红 | ✅ | ❌ |
| 境外资本利得 | ✅ | ❌ |
| 境外租金 | ✅ | ❌ |
| 退休金 / 年金（海外发放）| ✅ | ❌ |
| 资本利得 / 股息 | ✅ | ❌ |

> 💡 关键：**FEIE 仅适用"Earned Income"**（劳动所得），被动收入只能用 FTC。

### 居住要求对比

| 工具 | 要求 |
|------|------|
| **FTC** | **无居住要求**（在美国居住也可用）|
| **FEIE** | 必须满足**任一**： |
|  | a. **Bona Fide Residence**（海外"真实居民"整个税年）|
|  | b. **Physical Presence Test**（12 个月内在海外 ≥ **330 天**）|

**Physical Presence Test 的严格性**：
- 330 / 365 天 = **不到美国过春节** + **不能长期商务出差美国**
- 任何一天在美国都计入"美国天数"
- 国际公海 / 飞机过境 → 算海外

**Bona Fide Residence 的复杂性**：
- 需证明"真正居民"（不只是长期访客）
- 有当地工作合同、住房、家庭定居
- IRS 审查涉及意图、生活方式、经济联系
- 通常需要整个 **税年**（不是任意 12 个月）

### 决策矩阵

**按收入类型 + 税率 + 居住地**：

| 场景 | 推荐 |
|------|------|
| **海外工作，工资 < $132k，低税国**（新加坡、香港）| **FEIE**（完全排除，无需外国税抵）|
| **海外工作，工资 < $132k，中税国**（中国、欧洲）| 视税率：若外国税 < FEIE 节税 → FEIE 优 |
| **海外工作，工资 > $132k，高税国**（中国 45% 顶格）| **FTC**（排除后仍有剩余；FTC 可抵更多）|
| **海外工作，工资 > $132k，混合** | **FEIE + FTC 组合**（$132.9k 以下 FEIE，超出 FTC）|
| **美国居住 + 中国被动收入**（大多数回国者场景）| **只能 FTC**（FEIE 要求海外居住）|
| **中国居住 + 美国退休账户提取** | 复杂：FTC + 协定 Re-source |

### 组合策略（高收入海外工作者）

**FEIE + FTC 组合**：

```
海外工资：$200,000
外国税：$40,000（20% 假设）

Step 1：FEIE 排除 $132,900（2026）
  剩余应税：$200k - $132.9k = $67,100
  
Step 2：FTC 抵扣剩余部分
  美国税 on $67.1k ≈ $10,000
  外国税对应部分：$40k × ($67.1k / $200k) = $13,420
  FTC 抵扣：min($13,420, $10,000) = $10,000

净美国税：$0 （FEIE + FTC 全部抵消）
结转 FTC：$13,420 - $10,000 = $3,420（后递 10 年）
```

**注意**：
- FEIE 部分对应的外国税 **不能**做 FTC（已被"排除"，视为"未纳入"美国税）
- 外国税要按比例分配

### 重大陷阱：FEIE 的长期影响

**陷阱 1：退休账户供款**
- 用 FEIE 排除的收入 → **不算 Earned Income**
- **不能供 Traditional / Roth IRA**（需 earned income）
- FEIE 全额排除的人：**IRA 供款资格为零**
- FTC 路径：全部收入算 earned → 可供 IRA

**陷阱 2：EIC（Earned Income Credit）**
- FEIE 使用者不能领 EIC
- 低收入家庭可能失去补贴

**陷阱 3：Housing Exclusion / Deduction 叠加规则**
- FEIE 搭配 Housing Exclusion，但有"住房限额 × 地点因子"
- 中国一线城市（北京、上海、深圳、香港）有较高限额
- 其他中国城市限额低

**陷阱 4：撤销后 5 年禁区**
- 一旦做了 FEIE Election
- 撤销后 **5 年内不能重新选 FEIE**
- 除非 IRS 特批
- 策略错误可能锁定 5 年

### Social Security / Medicare 税

- FEIE 排除后 → 工资仍需缴 SS / Medicare（若雇主在美国）
- FTC 不影响 SS / Medicare 缴纳
- **与所得税**是独立议题

### 选择的动态因素

**何时切换 FTC → FEIE**：
- 外国工资降到 < $132k 且外国税率 < 美国税率
- 退休后不再有 earned income（FEIE 不适用）
- 需要供 IRA → 切换到 FTC

**何时切换 FEIE → FTC**：
- 外国工资涨到 > $150k + 高税国
- 想供 Roth IRA / Traditional IRA
- **注意 5 年禁区**（需 IRS 特批才能改）

### 实务建议（四种典型路径）

#### 路径 1：海外低税国派驻（< 5 年）
- **用 FEIE**
- 简单高效
- 接受不供 IRA 的限制

#### 路径 2：海外高税国长居（> 5 年）
- **用 FTC**
- 可继续供 IRA
- FTC 结转管理

#### 路径 3：海外混合（工资 + 被动收入）
- **FEIE 工资 + FTC 被动**
- 两份 Form（2555 + 1116）
- 专业 CPA 推荐

#### 路径 4：退休回国（本人典型场景）
- **只能 FTC**（FEIE 要求 earned income）
- 退休金、401K、IRA、SS 都是被动
- 严格执行 [[中国6年规则]] 避免中国全球征税
- 触发后 FTC 作兜底

### 中国场景专项

**场景 A：中国大公司派往美国（L-1）**
- 在美国是 RA → 非海外居住
- **FTC 适用**（抵扣美国税）
- FEIE 不适用（不是海外居住）

**场景 B：美国公司派驻中国（Expat）**
- 海外 330+ 天
- 工资 $150k，中国税 $25k
- **FEIE + FTC 组合最优**

**场景 C：中国自雇 / 小企业主（海外）**
- 海外居住
- 自雇收入
- **FEIE 适用**（但仍需 SE 税 15.3%）
- 或 FTC（若中国税高）

**场景 D：回国退休（保留绿卡）**
- 美国 401K 提取
- 中国（若第 7 年触发）征税
- **仅 FTC 适用**
- 可能通过协定 Re-source 抵免

### 对你个人（退休回中国）的结论

基于你的情况：
- 保留绿卡 → 永久美国税务居民
- 61 岁退休后主要收入是**退休金 + 401K 提取 + 美股分红**
- 都是 **Passive Income**（非 Earned）
- **FEIE 不适用**
- **FTC 是唯一工具**

**关键**：严格管理 [[中国6年规则]]（每年 30 天连续离境）→ 中国不征美国源 → FTC 也不太需要。若触发 → FTC 作兜底，但可能仍有净多交。
