## 定义

**PFIC 三种税务选择（Election）**：持有 [[PFIC]] 的美国人在三种规则下二选一（Default 不选也算选）：**Default（Excess Distribution）/ MTM（Mark-to-Market）/ QEF（Qualified Electing Fund）**。选择决定 5-10 年后的有效税率差 **30-60 个百分点**。**首年选择是永久性的**（除特殊 purging），错选代价极高。

## 核心要点

### 三种选择对比速查

| 维度 | Default（Excess Distribution）| MTM（Mark-to-Market）| QEF（Qualified Electing Fund）|
|------|---------------------------|----------------------|---------------------------|
| **选择条件** | 不做任何选择 | 可交易股票（公开上市）| 基金提供 AIS 报表 |
| **资本利得** | 惩罚性税率 | 普通所得率 | **保留长期优惠** |
| **未实现利得** | 无税（延迟）| 每年按市价 | 按 pro-rata |
| **分红** | 超过 125% 平均为"Excess" | 普通所得 | 普通所得 or 资本利得 |
| **利息罚款** | ✅ 有（Retroactive Interest）| ❌ 无 | ❌ 无 |
| **有效税率**（长期）| **50-80%** | 30-40% | **20-30%** |
| **合规复杂度** | 极高 | 中 | 中（若有 AIS）|
| **中国基金实用性** | ⚠️ 几乎强制默认 | 若上市可交易则可选 | 几乎不可行（无 AIS）|
| **首年必做** | ❌（默认）| ✅ 首年 or Purging | ✅ 首年 or Purging |

### 1. Default（Excess Distribution Regime）

**机制**（最复杂）：

```
第 1 年：买入 PFIC，无分红
第 2-4 年：少量分红累积
第 5 年：大额分红 或 卖出
  ↓
过去 3 年平均分红 × 125% = 基准
当年实际分红 - 基准 = Excess Distribution（若正）
卖出时资本利得 → **全部**视为 Excess Distribution

Excess Distribution 处理：
  → 按**持有全部年数**平均分摊
  → 每年部分按**那年最高普通税率**（2026 联邦 37%）
  → **加利息罚款**：按 IRS 利率（约 8%）从那年到今年复利
```

**算例**：持有 5 年，第 5 年卖出利得 $100,000（Default 规则）

```
$100,000 Excess Distribution 分摊到 5 年：
  年 1：$20,000 × 37% + 利息（5 年复利）= ~$10,800
  年 2：$20,000 × 37% + 利息（4 年）= ~$10,000
  年 3：$20,000 × 37% + 利息（3 年）= ~$9,200
  年 4：$20,000 × 37% + 利息（2 年）= ~$8,500
  年 5：$20,000 × 37%（当年）= $7,400
  
合计税 + 利息 = **~$45,900（45.9%）**
```

**额外问题**：
- 利息罚款**不可抵扣**
- 无法用 FTC 抵消中国已交税
- 资本利得本来在美国可 0%/15%/20%（长期）
- Default 抹除所有优惠

**何时会被迫 Default**：
- 不知道要做选择（95% 自然人用户）
- 错过首年 MTM / QEF 选择窗口
- 基金无 AIS 报表

### 2. Mark-to-Market（MTM）Election

**机制**：

```
买入年：选择 MTM（填 Form 8621 Part III）
每年：
  年末价值 - 年初价值 = 未实现利得（或损失）
  未实现利得 → 按**普通所得**课税（无长期优惠）
  未实现损失 → 可抵扣，但限制：
    - 抵扣上限 = 过去 MTM 已报的净利得
    - 不能抵扣超过净"累计"应税
卖出年：按 MTM 规则处理剩余
```

**算例**：持有 5 年，每年市值变化

```
年 1：$100k → $110k，+$10k 普通所得税 (37%) = $3,700
年 2：$110k → $105k，-$5k 可抵 = -$1,850 退税
年 3：$105k → $120k，+$15k → $5,550
年 4：$120k → $130k，+$10k → $3,700
年 5：$130k → $150k 卖出，+$20k → $7,400

合计税：$18,500（相比 Default $45,900 省 60%）
```

**MTM 的优势**：
- 无 Excess Distribution 爆雷
- 无利息罚款
- 税负平均分布在每年
- 可预测

**MTM 的劣势**：
- 未实现利得也要税（预付税）
- 无长期资本利得优惠
- 只适用**可交易股票**（公开市场能买卖）
- 中国某些非上市基金**不能**做 MTM

**MTM 的适用条件**：
- PFIC 是**可交易股票**：
  - 美国证券交易所上市
  - 外国合格交易所上市
  - 大多数 **公开发行的外国 ETF** 算
- 中国 A 股 ETF 上市在上交所/深交所 → **通常可做 MTM**
- 中国公募基金（非 ETF 型）→ **可能不符合**

**"首年"规则（关键）**：
- **首次取得 PFIC 的年份**做 MTM Election
- 否则需做 **Purging Election**：
  - 先按 Default 清算过去年份
  - 再从当年开始 MTM
  - 复杂 + 一次性 Default 税

### 3. QEF（Qualified Electing Fund）Election

**机制**（理论最优）：

```
基金提供 Annual Information Statement（AIS）
  ↓
股东按 pro-rata 比例报告：
  - Ordinary earnings → 普通所得
  - Net capital gain → **长期资本利得**（保留 15%/20% 优惠）

每年按 Form 8621 Part II + V 申报
无 Excess Distribution，无利息罚款
```

**算例**：持有 5 年，假设基金年利润 8% 来自资本利得

```
年 1：利润 $8k（资本利得）× 15% = $1,200
年 2-5：同样 → 每年 $1,200
卖出年：基础差额 → 长期资本利得
  $40k 额外利得 × 15% = $6,000

合计税：$6,000 + $4,800 = **$10,800（10.8%）**
```

**QEF 的优势**：
- 保留长期资本利得优惠
- 本质上把 PFIC 变成"美国合伙企业"
- 税负接近美国基金

**QEF 的致命问题**：
- **需要基金提供 AIS**（中国基金**几乎不提供**）
- AIS 要求按 IRS 口径的**年度盈余 + 资本利得拆分**
- 需基金配合（翻译 + 重算 + 法律合规）
- 中国公募基金不做此服务（市场太小，成本不值）

**QEF 在中国的实际情况**：
- **< 5% 的中国基金**可能有 QEF 报表（通常是香港互认基金或少数 QDII）
- 即使有，数据质量参差不齐
- 美国 CPA 需要独立 review
- **99% 中国基金 QEF 不可行**

**何时 QEF 可行**：
- 香港的某些国际基金（受美国投资者影响发 AIS）
- 开曼 / BVI 的合伙制基金（主动做 AIS）
- 中国少数"涉外"基金尝试提供
- 多需专业 CPA 确认

### 三种选择的决策树

```
Q1：基金是否提供 AIS（Annual Information Statement）？
  是 → Q2
  否 → 跳到 Q3

Q2：AIS 质量是否可信？CPA 是否能 review？
  是 → **QEF Election**（最优）
  否 → Q3

Q3：PFIC 是否"可交易股票"（公开上市）？
  是 → Q4
  否 → Default（唯一选择，尽快卖出）

Q4：是否是买入的第一年？
  是 → **MTM Election**
  否 → Q5

Q5：愿意做 Purging Election（一次性 Default 清算过去）吗？
  是 → MTM（之后年份）
  否 → Default（继续承受）
```

### 首年选择的重要性

**首年 = 取得 PFIC 的第 1 个税年**（不管多久后才报）：
- 未做 MTM / QEF → 默认 Default
- 后悔想改 → 需 Purging（先按 Default 清算）

**Purging 的代价**：
- 假设持有 5 年，市值从 $100k 涨到 $150k
- Purging 日视为**全部卖出**
- Default Excess Distribution 规则处理 $50k 利得
- 税 + 利息罚款可能 $20-25k
- 之后才能 MTM / QEF

**建议**：**首次发现 PFIC 立即咨询 CPA**，尽可能首年就做 MTM（若可交易）。

### 实务情景分析

**情景 A：中国公募基金（非 ETF）**
- 不是可交易股票 → **MTM 不适用**
- 无 AIS → **QEF 不适用**
- 只能 **Default**
- 建议：**尽快卖出**，不留

**情景 B：A 股 ETF（如 510300）**
- 在上交所上市 → 可能符合"可交易股票"
- 首年可做 **MTM**
- 未卖出每年按 MTM 报

**情景 C：香港互认基金**
- 部分基金提供简化 AIS
- 尝试 **QEF**，若不可行改 MTM

**情景 D：余额宝 / 零钱通**
- 货币基金，极难做 MTM（净值恒为 1 元）
- 唯一选择：**Default + 尽快清仓**

**情景 E：储蓄型保险（分红险、万能险）**
- 保单性质 → 不是"股票"
- 不能 MTM
- 不提供 AIS → 不能 QEF
- 只能 Default → **极糟**

### 对长期规划的影响

**"永远不买中国基金"的价值**：
```
避免 10 年持有 $100k 中国基金：
  Default 情况多交税：~$30-45k
  合规费用：$5-10k（10 年累计）
  心理负担 + 合规风险
  
节省：$35-55k 税 + 合规费
```

**回国前**：
- 清理所有 PFIC（Default 最后一次）
- 避免未来年度 Form 8621
- 若保留绿卡 → 美国 ETF 继续
