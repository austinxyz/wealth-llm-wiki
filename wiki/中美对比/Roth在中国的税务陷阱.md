## 定义

**Roth 免税性在中国可能失效**：美国 Roth IRA / Roth 401K 提取在**美国免税**，但中国不承认 Roth 账户的免税性质。若退休回中国且触发中国全球征税（[[中国6年规则]]第 7 年），Roth 提取可能按中国个税 **20% 被动所得**或普通所得税率征收。这是跨境退休规划**最反直觉的陷阱**——美国的最优账户在中国可能变成次优。

## 核心要点

**美国 Roth 的税务定位**：
- 供款时已纳税（after-tax）
- 增长免税
- **满足条件提取**（59.5 岁 + 5 年规则）→ **完全免税**
- 美国联邦 + 州 + NIIT 全部为零

**中国的税务认知（2019 新个税法后）**：
- 中国**不承认**美国 Roth 账户的免税性质
- 中国只看**分配行为**是否应税
- Roth IRA 分配 → 按中国税法归类

**关键判定：Article 17 vs 普通所得**：

中美税收协定 Article 17（Pensions）对**政府养老金以外**的**私人养老金**：
- 协定允许**仅由居住国征税**
- 若成功援引 → 美国可能免征（已本来免税）+ 中国按其规则征

**问题**：Roth IRA 是否算协定下的"Pension"？
- **IRS 观点**：Roth IRA 是退休账户，属于 Pension 范畴
- **中国税局实践**：**不明确**，部分税务官员将其视为"境外财产性收入"
- **实务结果**：可能按**20% 被动所得**或**45% 顶格综合所得**征收

**不同场景下的税务后果**：

| 场景 | 美国税 | 中国税 | 净效果 |
|------|-------|-------|-------|
| 美国居民（未回国）| $0 | N/A | **完全免税 ✅** |
| 回国 6 年内 + 每年 30 天离境 | $0 | **$0**（仅中国源征税）| **完全免税 ✅** |
| 回国第 7 年 + 全球征税 | $0 | **20% 或 45%** | **部分或完全失去免税** ❌ |
| 放弃美国身份后回国 | 按 NRA 规则（少量美国源）| 20% 或 45% | Roth 优势完全丧失 |

**算例**：回国第 7 年提取 $100k Roth IRA

```
美国端：
  $0 税（Roth 免税提取）

中国端（保守估算）：
  方式 A：按被动所得 20%
    = $100,000 × 20%
    = $20,000 税
  
  方式 B：按综合所得累进（45% 顶格）
    = $100,000 落入 ¥680k 以上档
    = ~$45,000 税

有效损失：$20,000 - $45,000（相比美国居民期间提取）
```

**对比 Traditional IRA 提取**：

```
美国居民时提取：
  - 美国税：22% × $100k = $22,000
  - 最终拿到：$78,000

回国第 7 年提取：
  - 美国税：22% × $100k = $22,000
  - 中国税：可能 20% = $20,000（扣抵 $22,000 美国税后 = 0）
  - FTC 抵：$20,000
  - 净税：$22,000（美国）
  - 最终拿到：$78,000

→ Traditional 回国提取**无额外损失**（FTC 抵）
```

**关键反转**：
- **Roth**：美国免税 → 回国可能全额被中国征
- **Traditional**：美国征税 → FTC 可抵中国端
- 在**极端情况下**，Traditional 回国提取**比 Roth 优**！

## 规划建议

### 策略 1：回国前最大化 Roth 提取

**执行**：
- 退休到回国前的窗口期（2-5 年）
- 如 61 岁退休、65 岁回国 → **4 年窗口**
- 每年提取 Roth 数万至数十万
- 资金进入 Taxable 账户
- 回国时 Roth 已清空 → 无后续中国税风险

**例**：
```
Roth 账户 $500k
4 年退休窗口，每年提 $125k
→ 回国时 Roth = $0
→ 无论中国税怎么对待，已不影响
```

### 策略 2：不做 Roth Conversion（反常识）

**常规建议**：退休低收入年做大量 Roth Conversion
**回国场景**：若确定回国 + 7 年后触发全球征税

```
Conversion 的效果：
  美国：+22% 税（当年），未来免税
  中国：不认 Roth → 未来提取仍被中国征

→ 总效果：多交了美国税，没省中国税
→ 不如保留 Traditional，用 FTC 抵
```

**决策矩阵**：

| 退休地 | Roth Conversion 建议 |
|-------|------------------|
| 美国养老 | ✅ 尽量转，锁定美国免税 |
| 中国养老（6 年规则管理得当）| ✅ 仍建议转，中国端不征 |
| 中国养老（第 7 年必触发）| ❌ **慎重**，可能得不偿失 |
| 中国养老 + 放弃美国身份 | ❌ 已无美国税优势 |

### 策略 3：[[中国6年规则]]管理

**核心**：确保每年一次 **连续 30 天离境** → 6 年计时重置 → **永不触发**第 7 年全球征税

**若成功**：
- Roth 提取**仅中国源征税**（美国 Roth 是境外，不征）
- **完全保留 Roth 免税性** ✅

**执行要点**：
- 每年夏天 / 冬天出国 30+ 天（连续）
- 留意：累计不算，必须连续单次
- 保留护照 + 机票证据

### 策略 4：通过 Annuity 包装

**机制**：
- 美国 Roth → 转换为 Annuity（年金）
- 中美协定 Article 17 对 Annuity 可能有**更明确**的"仅居住国征税"规定
- 协定保护可能优于裸 Roth 提取

**限制**：
- Annuity 流动性差（锁定期 + 罚金）
- 费用高（管理费 1-3%）
- 需专业规划

### 策略 5：Roth IRA 本金 vs 增长分开

**美国 Roth 提取规则**：
- 本金（contribution）随时可取（无税无罚）
- 增长（earnings）需 5 年 + 59.5 岁

**中国视角**：
- 可能将本金视为"已税资本回归"（不征）
- 增长按被动所得征

**执行**：
- 回国前先取完增长部分（美国免税）
- 回国后仅取本金 → 中国可能不征

### 策略 6：Roth → 慈善捐赠

**Qualified Charitable Distribution（QCD）**：
- 70.5 岁 + Traditional IRA → 直接捐给慈善机构
- 金额抵 RMD + 不计入 AGI
- Roth 无 RMD，但可直接捐
- 中国公益机构**不符合** IRS 501(c)(3) → 不享受 QCD

**中国本地捐赠**：
- 中国对慈善捐赠有税前扣除
- Roth 分配后用于中国捐赠 → 中国端可扣除（部分）
- 减轻中国端征税压力

## 实务决策树

```
Q1：确定长期回中国养老吗？
  否 → 正常 Roth 策略（最大化供款 + Conversion）
  是 → Q2

Q2：能保证每年 > 30 天连续离境吗？
  能 → [[中国6年规则]]管理 → Roth 安全，正常策略
  不能 → Q3

Q3：第 7 年触发不可避免？
  是 → 规划回国前 3-5 年提完 Roth
  否 → 继续监控

Q4：打算放弃美国身份？
  是（LTR）→ 见 [[Exit-Tax]]，Roth 提取规则改变
  否 → FTC 规划
```

## 常见误解

- ❌ "Roth 是免税账户，中国也会认可" → 错，中国不承认免税性质
- ❌ "用 Form 8833 援引协定 = 万能" → 协定下 Pension 定义不明
- ❌ "只要不取出就没事" → 死亡时大量未提取 Roth → 受益人（可能 [[Section-2801遗产]] 40% 税）
- ❌ "Roth Conversion 永远最优" → 回国场景可能次优
- ❌ "中国税 < 美国税，多交点没关系" → Roth 原本免税，多交的是**纯损失**
